# 古风 / 穿越 / 重生古代 爆点全集

> 2026 短剧古风题材热度榜：《烧火丫头闹侯府》《凤还朝》《将门捉婿》《大雍第一狂婿》《一品布衣 4》等霸榜。古风偏稳定，套路成熟。

## 一、嫡女归来 / 重生宅斗（古代版重生复仇）

### 🎬 核心公式
**前世被庶妹/继母/夫君害死 → 重生回闺中时代 → 用前世记忆反杀全家**

### 📦 经典桥段

1. **前世被毒酒/绣花针/产难/火灾害死**
2. **重生回到嫁人前一天**：抢救一切的最后机会
3. **退掉烂亲事**：第一波打脸
4. **第二次嫁人嫁给世子/王爷**：黄金男主
5. **庶妹假装柔弱实则恶毒**：标配反派
6. **嫡母装贤良实则毒辣**：双线反派
7. **绣花 / 弹琴 / 厨艺 / 医术 一鸣惊人**
8. **以一己之力撑起娘家**：娘家衰败戏码
9. **滴血认亲 / 验毒 / 验身世** 高潮反转
10. **皇家恩宠**：太后赐婚 / 皇帝御赐

### ⚠️ 烂梗

- ❌ 反派全员恶毒，没有任何复杂动机
- ❌ 嫡女永远是"白纸 + 全能"，缺少瑕疵
- ❌ 男主永远是冷面王爷或腹黑世子

### ✨ 微创新

1. **嫡女重生后选择不嫁**：搞事业、做生意、闯天下
2. **嫡女与庶妹和解**：庶妹也是受害者
3. **男主才是穿越/重生者**：男主视角

## 二、长公主 / 大女主古风（《长公主在上》《凤还朝》类型）

### 🎬 核心公式
**女主身份显赫（长公主/皇后/摄政女王）→ 朝堂博弈 → 帝后双权**

### 📦 经典桥段

1. **太后倒台 / 摄政王篡位 / 皇帝懦弱** 三选一开局
2. **长公主一人撑朝堂**：女强戏标志
3. **太子是傀儡**：长公主辅政
4. **暗卫 / 影卫 / 镇北王**：男主标配身份
5. **女主带兵打仗**：金戈铁马戏
6. **朝堂辩论 / 殿试 / 立储** 智斗戏
7. **联姻政治**：女主嫁敌国王子
8. **女主黑化 / 上位**：高潮反转
9. **女主与皇帝兄妹 / 姑侄 / 母子情**：情感主线
10. **江山美人选江山**：女主向

### ⚠️ 烂梗

- ❌ 朝堂戏全靠女主"机智"金句开挂
- ❌ 男主必备"我助你登基"剧情已经俗烂
- ❌ 政治戏写成宫斗戏

### ✨ 微创新

1. **女主不要江山要离开**：反向选择
2. **男主才是傀儡**：女主真正掌权
3. **女主与女主之争**：太后 vs 长公主，女女博弈
4. **女主选择联姻不是被迫，是主动**：政治家觉悟

## 三、穿书 / 穿越（女主穿成炮灰/反派）

### 🎬 核心公式
**女主穿成书中炮灰 → 知道剧情走向 → 反向操作改命**

### 📦 经典桥段

1. **穿成原书反派**：女配/恶毒后娘/恶毒嫂子
2. **穿成炮灰女配**：原本要死的命
3. **穿成女主的对照组**：男主原本爱原女主，现在爱穿越的"反派"
4. **作者大大救我**：和作者交流的脑洞
5. **角色觉醒反抗作者**：纸片人觉醒
6. **多人穿书**：原女主也是穿的
7. **男主察觉剧情不对**：男主才是真"穿越者"
8. **改命任务系统**：完成 XX 任务可回原世界

### ⚠️ 烂梗

- ❌ 主角"剧情说"内心独白太多
- ❌ 原女主纯绿茶
- ❌ 男主第一眼就爱上"反派"——缺过程

### ✨ 微创新

1. **穿成的角色其实是个好人**，被作者诬陷
2. **穿越者反而被原住民玩弄**：剧情智斗
3. **多人穿书 + 角色觉醒**：群像
4. **男主第一眼讨厌"反派"**：慢热版

## 四、烧火丫头逆袭（《烧火丫头闹侯府》验证爆款）

### 🎬 核心公式
**女主出身底层（丫鬟、烧火、农女）→ 偶然救助落魄世子 / 被卷入豪门 → 用厨艺/医术/聪明逆袭**

### 📦 经典桥段

1. **女主烧火 / 喂猪 / 砍柴的开局**：极致接地气
2. **男主重伤倒在女主门口**：拉郎配开局
3. **男主在女主家养伤**：日久生情
4. **男主身份揭开**：世子 / 太子 / 王爷
5. **女主用现代知识改良美食 / 农具 / 算账**
6. **男主带女主进京**：身份反差戏
7. **侯府/王府的歧视**：嫡子嫡女、妾室全面打压
8. **女主烤肉 / 包子 / 火锅 / 烤鸭一鸣惊人**：吃货流
9. **男主反对父母强迫婚姻，誓死要娶女主**
10. **女主自立门户**：开酒楼 / 开医馆 / 开布庄

### ⚠️ 烂梗

- ❌ 女主一来就会做现代美食，反差太突兀
- ❌ 男主家人一开始全反对 → 后期一秒接纳
- ❌ 女主厨艺单一（烤肉、串、火锅）已被写烂

### ✨ 微创新

1. **女主厨艺是真传承**，不是开金手指
2. **女主独立后甩开男主**：男主追妻
3. **女主带着童年伤痕**：心理弧线
4. **男主家庭不是反对，而是支持**：双向奔赴版

## 五、王爷追妻 / 帝王追妻

### 🎬 核心公式
**男主曾辜负女主 → 女主逃离 / 死过一次 → 男主追悔莫及**

### 📦 经典桥段

1. **王爷宠妾灭妻**：女主前世惨死
2. **女主下堂 / 自请离开**：心冷退场
3. **女主嫁给敌国王 / 江湖侠客 / 邻国太子**：男主炸毛
4. **女主带着孩子归来，孩子不是男主的**：钩子王
5. **男主跪在城门 / 雪地 / 凤栖殿** 求复合
6. **白月光真相曝光**：男主才知她是坏的
7. **女主已封后 / 封凤 / 称帝**：身份反转
8. **男主疯了 / 病了 / 失踪十年**：苦戏

### ⚠️ 烂梗

- ❌ 男主前 5 集对女主身体暴力 → 已被监管限制
- ❌ 白月光必坏 + 主母必善 → 太脸谱

### ✨ 微创新

1. **王爷追妻不是因为爱，是因为利益**：政治算计反转
2. **女主追王爷**：颠倒性别
3. **王爷追妻同时女主在追江山**：双线
4. **白月光不坏，但就是真心不喜欢王爷**
