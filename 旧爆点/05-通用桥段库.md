# 通用桥段库（所有题材都能用的"零件库"）

> 这是一份"模块化桥段"目录。无论你写什么题材，把这些桥段当作可拼装的乐高积木。
>
> ⚠️ 标记 🔴 的是已经"用烂"的桥段，慎用；标记 🟡 是"半烂"，可微创新；🟢 是"还能打"的桥段。

## 一、开场钩子（前 300 字必出）

### 🔴 已被用烂的开场

1. 🔴 "我重生了，回到了……"
2. 🔴 "嫁给他三年，他从未正眼看过我"
3. 🔴 "离婚协议书摆在桌上"
4. 🔴 "我以为他爱我，直到我看见……"
5. 🔴 "全家就我一个人住地下室"
6. 🔴 "婆婆又来作妖了"
7. 🔴 "闺蜜抢了我的男朋友"

### 🟡 半烂的开场（需要变体）

8. 🟡 "我死了，死在我最爱的人手里"
9. 🟡 "婚礼上，新娘不是我"
10. 🟡 "他抱着白月光的那一刻，我决定离婚"

### 🟢 当下高分的开场公式

**公式 A：荒诞行为 + 极端后果**
- "她在业主群说要把全楼门锁改成她的生日"（你的范文 1）
- "新人说我赚他们黑心钱，换了 200 块的装备后摔成骨折"（你的范文 2）
- "前任在婚礼当天宣布我得了不孕症"
- "婆婆要我用工资换她儿子的房产证"

**公式 B：信息差直接抛**
- "他不知道，我才是这家公司的真老板"
- "他们以为我是个废物，包括我妈"
- "她不知道，那把推她下楼的椅子是我准备的"

**公式 C：未来视角倒叙**
- "三个月后，她会跪在我面前求我别死"
- "如果不是那天的电话，我可能现在还在装乖"

**公式 D：群体围攻 + 主角冷静**
- "整个楼层的人都在说我冷血"
- "全班同学都让我滚出去"

### ✨ 开场创新方法

1. **第一句必须有"动作 + 反差"**：不要描写心情，要描写正在发生的事
2. **前 100 字内出现 2~3 个人物**：让冲突立刻产生
3. **前 200 字内必须出现"利益冲突"**：钱、权、爱、命，四选一
4. **第 300 字一定要有一个"小钩子"**：让读者明白"接下来会发生大事"

### 打脸场景的"三幕剧"结构（标准化模板）

```
【第一幕 · 压制】（占 30% 篇幅）
- 反派出招：贬低、辱骂、栽赃
- 群体跟随：周围人附和
- 主角隐忍：不解释、不愤怒、表面接受
- 关键标志：让读者气得想骂人

【第二幕 · 转折】（占 20% 篇幅）
- 触发事件：电话来 / 证据出 / 大人物到
- 反派愣住：意识不对
- 信息差打开：真相要被揭开
- 关键标志：让读者屏住呼吸

【第三幕 · 碾压】（占 50% 篇幅）
- 身份揭露 / 实力展示 / 反派被怼
- 群体反水：刚才附和的人开始巴结主角
- 反派下场：物质损失 + 名誉社死 + 法律追责（三连击最爽）
- 关键标志：让读者爽到拍腿
```

### 反转类型

#### A. 身份反转
1. 路人 → 大佬
2. 普通员工 → 真老板
3. 弱者 → 隐藏强者
4. 假千金 → 真千金
5. 替身 → 真身

#### B. 关系反转
6. 闺蜜 → 仇人
7. 仇人 → 救命恩人
8. 路人 → 哥哥/亲人
9. 死敌 → 同盟
10. 老婆 → 老板的女儿

#### C. 时间反转
11. 现在 → 重生过去
12. 当下 → 未来视角
13. 现实 → 梦境 / 元宇宙
14. 这一世 → 上一世
15. 倒叙 → 真相重置

#### D. 因果反转
16. 你以为是 A 的错，其实是 B
17. 你以为是巧合，其实是设局
18. 你以为是好心，其实是恶意
19. 你以为输了，其实赢了
20. 你以为赢了，其实进了陷阱

#### E. 道具反转
21. 假药 / 假酒 / 假证（你的范文 2：假装备）
22. 真假鉴定
23. 监控录像
24. 录音 / 聊天记录
25. 银行流水 / 财务造假

### 反转的"埋伏笔三段式"

```
1. 第一次出现：看似无意提及（如：女主在公司"打杂"）
2. 第二次出现：被反派当作攻击点（如：反派嘲讽"你就是打杂的"）
3. 第三次出现：真相曝光（如：女主其实是 CEO）
```

⚠️ **避免硬反转**：反转必须有伏笔，否则读者觉得是"作者强行操控"。

## 四、章末钩子库（追读率的命门）

### 钩子类型

| 类型 | 示例 | 效果 |
|---|---|---|
| 突发事件 | "话音未落，门被踹开" | 紧急感 |
| 真相揭露 | "她转身的瞬间，他看见了那道疤" | 信息感 |
| 误会加深 | "他冷笑：原来你早就计划好了" | 张力感 |
| 反派出招 | "三天后，警察出现在她门口" | 危机感 |
| 暧昧推进 | "他突然把她按在墙上" | 心跳感 |
| 隐患浮现 | "她不知道，门外有双眼睛在看着" | 不安感 |
| 实力升级 | "她抬起手，那枚戒指开始发光" | 期待感 |
| 反派翻车 | "他打开包裹，脸色一下白了" | 爽感 |
| 关键决定 | "她拿起笔，在离婚协议上签下名字" | 节点感 |
| 时间锁定 | "距离比赛只剩 24 小时" | 倒计时 |

### 钩子的"3 大忌"

- ❌ 钩子和下一章没有关系
- ❌ 钩子太频繁（每章都是"突发"会让读者麻木）
- ❌ 钩子是假的（虚晃一枪，下一章被回避）

## 五、人物配置库

### 主角的"人设三板斧"

每个主角必备三个特质：

1. **核心动机**（他/她想要什么？）
2. **致命弱点**（他/她怕什么？）
3. **反差特质**（让人物立体的"反预期"）

举例：
- 古武大佬主角，但**爱吃糖**怕苦——反差萌
- 顶级 hacker 女主，但**社交恐惧**——反差点
- 战神归来男主，但**特别怕老婆**——反差爽

### 反派配置矩阵（4 种基本反派）

| 类型 | 特征 | 动机 | 经典代表 |
|---|---|---|---|
| **白莲花型** | 表面柔弱，背地心机 | 上位 / 抢资源 | 姜棠、陈依依 |
| **明刀型** | 表面强势，公开欺压 | 维护既得利益 | 婆婆、上司、长辈 |
| **跟班型** | 没有主见，只会附和 | 站队 / 蹭好处 | 业主群里的邻居们 |
| **隐藏型** | 表面盟友，关键时刻反咬 | 阴谋 / 嫉妒 | 闺蜜、丈夫、恩人 |

**好的反派必须有自洽利益逻辑**（2026 行业共识）：
- 周经理为什么帮姜棠？因为楼盘宣传 KPI
- 陈依依为什么坑攀岩队？因为她也是开店的，想抢生意
- 赵承泽为什么护着姜棠？因为他和姜棠有关系

### 配角组合的"标准 5 件套"

1. **闺蜜/兄弟（盟友）**：陪主角说话、推剧情
2. **导师 / 上司（资源提供者）**：关键时刻送装备
3. **暧昧对象（情感线）**：男 / 女主或男 / 女配
4. **死对头（次反派）**：日常制造冲突
5. **观众视角（路人）**：吃瓜群众，代表读者反应

## 六、对话设计库（这是网文最重要的部分）

### 网文对话的"4 大铁律"

1. **80% 对话占比**：手机阅读，眼睛不喜欢看长段落
2. **每句对话短**：超过两行的对话切成两段
3. **对话推进剧情**：不能闲聊，每句话都要有信息量
4. **对话带情绪**：用"冷笑"、"嘲讽"、"咬牙"、"颤抖"等动作标签

### 高频金句模板

```
"我以为你是 X，没想到你是 Y。"
"你 / 你们等着。"
"赵承泽，你别把话说绝。"
"这一次，我不会再救他们了。"
"你们想让我替你们认错，做梦。"
"睁开你的狗脑子想想清楚。"
"少废话！"
"你听见没有？X 到现在还替你说话。"
"她一个小姑娘，哪懂这些？"
"为什么总有人恶意揣测我？"
"啧啧，含泪赚我一千块？"
"……"（沉默就是金句）
```

### 反派的"6 大经典台词"

1. "你以为你是谁？"
2. "你不就是个 X 的吗？"
3. "X 棠/X 依只是开个玩笑，你为什么这么较真？"
4. "你这么做，对得起我们吗？"
5. "她只是太单纯了！"
6. "都是邻里 / 同学 / 同事，没必要闹这么难看。"

### 主角的"6 大反击句式"

1. "OK。"（最高级反击）
2. "你说得对，但与我无关。"
3. "这么巧？"
4. "我没兴趣。"
5. "随便。"
6. （直接挂电话 / 拉黑 / 退群）

## 七、情绪节奏模板

### 黄金节奏曲线（爽文标配）

```
压抑 → 反击 → 小爽 → 反派反扑 → 大爽
  ↓     ↓      ↓        ↓         ↓
 5%    15%    20%      40%       20%
```

### 章节内部的"起承转爽"

| 段落 | 占比 | 任务 |
|---|---|---|
| 起 | 15% | 引出本章冲突 |
| 承 | 35% | 冲突升级、压力增大 |
| 转 | 15% | 出现转折、敌人露底牌 |
| 爽 | 25% | 主角反击 / 真相揭晓 |
| 钩 | 10% | 章末钩子 |
