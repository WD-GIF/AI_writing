# 标题公式 · 爆款命名套路全集

> 网文 / 短剧 / 推文 / 公众号短篇的爆款率 **40% 取决于标题**。
> 你范文的标题就是典型样本：
> - "重生后，团宠邻居把全楼门锁改成她生日"
> - "攀岩队友说我赚他们黑心钱，换了 200 块的装备后摔成骨折"
>
> 这种标题在公众号 / 抖音推文场有专门的命名学。

---

## 一、爆款标题的 5 大核心要素

### 1. 信息密度极高
标题里要让读者**一句话看完一个故事**。
- ❌ "重生归来"（信息量 = 0）
- ✅ "重生后，团宠邻居把全楼门锁改成她生日"（信息量 = 4：重生 + 团宠邻居 + 全楼门锁 + 生日密码）

### 2. 必须有"反常识 / 反预期"
让读者第一反应是"卧槽这什么操作"。
- "把全楼门锁改成她生日" → 离谱
- "换了 200 块的装备后摔成骨折" → 报应

### 3. 必须有"动作 + 后果"
**主体 + 干了什么 + 结果如何**
- 团宠邻居 + 改门锁 + (隐含的事故)
- 攀岩队友 + 说我黑心 + 摔成骨折

### 4. 标题剧透高潮（反直觉但有效）
传统观念里"剧透 = 减分"，但短剧 / 推文是反过来的：
- 标题已经剧透"反派翻车" → 读者反而想看**翻车过程**
- 标题透出"主角复仇" → 读者反而想看**复仇细节**

### 5. 必须有具体数字 / 名词
具体细节比抽象描述更勾人：
- ❌ "她换了便宜装备出事了"
- ✅ "换了 **200 块** 的装备后摔成骨折"

## 二、12 种爆款标题公式

### 公式 1：荒诞行为 + 后果（你的范文 1）

```
[反派] [离谱行为]，[后果 / 隐含后果]
```

例子：
- 重生后，团宠邻居把全楼门锁改成她生日
- 婆婆把我工资卡改成她的支付密码后离婚了
- 老公给小三买了套房，没想到房产证写我名字
- 闺蜜偷我快递，里面是寄给我的离婚协议
- 重生后，前夫把抚养费汇到我闺蜜账户上

### 公式 2：诬陷 + 反转报应（你的范文 2）

```
[反派] 说我 [污蔑]，[反派自己出事 / 反转]
```

例子：
- 攀岩队友说我赚他们黑心钱，换了 200 块的装备后摔成骨折
- 同事说我靠脸上位，结果发现我才是大老板的女儿
- 婆婆说我吃软饭，结果我才是给她儿子发工资的人
- 闺蜜说我傍大款，结果她抢我"大款"破产了
- 邻居说我装清高，结果我家比她家贵 100 倍

### 公式 3：反派群体围攻

```
[反派群体] 一起 [对我做了什么]，[反转/我的反应]
```

例子：
- 全家都说我克夫，结果他们一个个出事
- 全公司都说我是关系户，结果老板是我亲爹
- 全班同学嘲笑我穷，校长发奖学金那天我成榜首
- 整栋楼业主联合起来逼我搬家，物业才知道我是房东

### 公式 4：重生 + 决心型

```
重生后/这一次，我决定 [反常规决定]
```

例子：
- 重生后，我决定不嫁人，专门种田
- 重生后，我决定让前夫和小三过到底
- 重生后，我决定先和闺蜜抢资源
- 这一次，我不会再救他们了

### 公式 5：身份反转型

```
[别人以为我是 X]，[其实我是 Y]
```

例子：
- 她以为我是来攀亲戚的，其实我才是首富的女儿
- 她以为我是路人甲，其实我是她公司的董事长
- 我以为我嫁了个穷小子，其实他是隐藏富豪
- 全楼业主以为我是钉子户，其实我是开发商

### 公式 6：错位对比型

```
[A 做了 X]，[B 却做了 Y]（强烈对比）
```

例子：
- 婆婆给小姑陪嫁 100 万，给我 100 块
- 男友给前任买 LV，给我买拼夕夕
- 我给他攒了 200 万存款，他给小三花了 199 万
- 我陪他熬到当总裁，他和秘书一起踢我出公司

### 公式 7：时间锁定 + 反转型

```
[X 时间内 / 之后]，[反转事件]
```

例子：
- 三年里我每天给他做饭，离婚那天他说我做的菜难吃
- 五年攒了 50 万首付，签合同那天发现房子写的他妈名字
- 婚礼当天，伴娘穿白色婚纱走红毯
- 离婚一年后，他求我复婚的方式是来要补贴

### 公式 8：金额具体型（推文场超有效）

```
[反派 + 具体金额 + 操作]
```

例子：
- 攀岩队友说我赚他们黑心钱，换了 200 块的装备后摔成骨折
- 婆婆要儿子给她 50 万，结果儿子刚把卡递过去她就拿去给小儿子
- 我借给闺蜜 30 万，她拿去做了我前男友的婚礼
- 公婆要我给小叔子付 80 万买房，我说先把婚前我借你们的还了

### 公式 9：剧情核心 + 高情绪触发词

```
[剧情] + [社死 / 报应 / 跪求 / 哭着 / 含泪]
```

高情绪触发词列表：
- **社死**、**翻车**、**塌房**、**爆雷**
- **跪求**、**痛哭**、**疯了**、**麻了**
- **报应**、**反杀**、**打脸**、**逆袭**
- **含泪**、**哭着**、**追悔莫及**

### 公式 10：第二人称型（公众号高频）

```
[反派] 你不会真以为 [离谱认知] 吧？
```

例子：
- 婆婆，你不会真以为我会忍你一辈子吧？
- 老公，你不会真以为家产都是你的吧？
- 闺蜜，你不会真以为我看不出来你抢我男人吧？

### 公式 11：群体围观 + 反派翻车

```
[全网/全公司/全班/全家] 都在 [围攻我]，结果 [反转]
```

例子：
- 全网都骂我冷血，警察上门那天他们都闭嘴了
- 全班同学都让我滚出去，校长开会那天我成了榜样
- 全家都觉得我是赔钱货，分家产时才发现我是私生女继承人

### 公式 12：反派得意 + 主角反手（你范文 1 的精华）

```
[反派] 自以为 [得意行为]，[主角] [反向操作让其翻车]
```

例子：
- 团宠邻居把全楼门锁改成她生日，结果半小时前我就关了系统权限
- 公婆得意地把我赶出门，没看见我手里的房产证
- 小三在直播间炫耀我老公的礼物，下一秒律师函送到她家
- 闺蜜抢走我的男友，没发现我早就把男友的资产转走了

## 三、平台差异化标题策略

### 番茄小说（章节书）
- **30 字以内**：手机列表展示
- **核心矛盾 + 钩子词**：例 "重生后我把全家虐成狗"
- **类型词前置**：[重生]、[团宠]、[追妻]、[穿越]
- **男频用"狠"字，女频用"虐"字**

### 抖音推文 / 红果短剧（关键 KOL 类型）
- **30~40 字**：信息浓缩
- **必带"反派损失"**：让读者预期"会爽"
- **常用词**：摊牌、社死、塌房、翻车、跪、爆

### 公众号爽文（单篇 1~3 万字）
- **40~60 字**：长标题反而效果好
- **必带数字 / 具体名词**：让标题有"画面感"
- **常用结构**："X 行为，Y 后果，Z 反转"

### 知乎盐选 / 短故事
- **疑问句 / 第一人称**：例 "丈夫给小三花了 199 万，最后他给我留了 1 万——这一万元怎么花？"
- **故事悬念前置**：让人想点开
- **避免直接剧透**：保留悬念是知乎特色

## 四、标题禁忌（写了基本不推）

1. ❌ **过度抽象**：例 "命运" "缘分" "人生" 这类词
2. ❌ **太短**：例 "重生" "复仇" 没有信息量
3. ❌ **太文艺**：例 "桃花落尽时" 不是网文
4. ❌ **谐音烂梗**：例 "婆婆 / 婆乐" "爱情 / 爱琴" 等
5. ❌ **多于 3 个标点符号**：例 "她！竟然！背叛！我！"
6. ❌ **过度使用问号**：例 "他真的爱我吗？？？"
7. ❌ **过度引用网红词**：例 "栓 Q" "尊嘟假嘟" 半年就过时
8. ❌ **平台违禁词**：例 "出轨" "性" 在番茄是禁

## 五、标题练习模板

挑一个你想写的剧情，按公式套：

```
公式：[反派][离谱行为]，[反转/后果]

剧情核心：婆婆把我儿子户口偷偷迁到小叔子名下
对应标题：
- 婆婆把我儿子户口偷偷迁到小叔子名下，离婚分家产时她哭着求我别走
- 婆婆偷迁孙子户口给小叔，分家产那天她跪在我面前
```
