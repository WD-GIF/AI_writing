# 《蓉颜小厨》专用风格指令包

> **目的**: AI 每次写《蓉颜小厨》前必读，强制对照执行  
> **来源**: 从 DramaBench 500 个剧本中检索出 15 部最相关样本，统计提炼  
> **特别参考**: `script_5431 Revenge of a Grieving Mother`（核心冲突几乎一致）

---

## 🔒 硬性指标（违反一条 = 重写）

### 1. 句长规则

| 类型 | 字数 | 占比目标 |
|---|---|---|
| 对白 | **平均 12-15 字**，中位数 12 | 全文 60% 在 5-15 字区间 |
| 长对白 (>30字) | 全文 ≤ 10% | 只能在"真情爆发"时出现 |
| 短对白 (<5字) | 8-10% | 关键打脸/反转时使用 |
| 动作描述 | **平均 20-30 字** | 一句话只描一个动作 |
| 旁白 OS（女主第一人称） | **10-25 字** | 单句不超 30 字 |

**禁忌**：连续 3 句以上都 >25 字 → AI 腔严重

### 2. 旁白 vs 对白配比

| 段落类型 | 旁白:对白 |
|---|---|
| 开篇 0-150 字 | 7:3（OS 钩子先抓人） |
| 矛盾展开 150-500 字 | 5:5 |
| 打脸高潮 500 字+ | 3:7（对白密集） |

### 3. 情绪标记（每个对白前/后允许 1 个）

**必须使用的 8 种**（来自统计 Top 8）：
1. （冷冷地）
2. （低语）
3. （轻声）
4. （颤抖）
5. （震惊）
6. （申辩）← 反派专用
7. （安静地）← 女主隐忍专用
8. （中断）← 暴力打断对话

**禁忌**：禁用"含泪地"、"愤怒地"、"温柔地"——太套路

### 4. 反派对白三类模板

**Type A - 道德绑架**（嫂子专用，仿 script_5431）：
- 句式："你 X，我只 Y，请 Z"
- 例："你弟弟在贾辉小的时候就死了，我只剩下他了，请你先检查一下"
- 移植到《蓉颜小厨》：
  - **嫂子**："念念已经那么大了，豆豆才两岁，你做妈的就不能让一让？"
  - **婆婆**："你弟妹这是为你好，你做嫂子的怎么这点眼力见都没有"

**Type B - 直接侮辱**（短促 ≤8 字，用于真翻脸时）：
- "滚出去!"
- "你疯了吗?"
- "你不配做妈"
- **禁忌**：不要写"赔钱货"这种农村剧台词

**Type C - 阴阳怪气**（妯娌专用，最常用）：
- 句式："X 啊，Y 嘛"
- 例："姐姐，这是珍贵的"
- 移植：
  - **蓉蓉**："嫂嫂啊，您这是上辈子修来的福气嘛，我可羡慕你"
  - **蓉蓉**："念念啊，你妈忙工作呢，咱们小婶给你做好吃的"

### 5. 女主 OS（旁白）四类模板

**Type 1 - 冷峻复仇宣言**（开篇必用）：
- "他应该失去一切。"
- "我会查出是谁干的，我发誓。"
- 移植：**"周文博，这一世，我要让你和蓉蓉，连同那个姓周的孩子，一起跪在我脚下。"**

**Type 2 - 痛悔自责**（占 30%）：
- "或许我活该。"
- "天堂还给了我这些日子。"
- 移植：**"念念，妈妈对不起你。上一世我以为爸爸会照顾你，可那是我这辈子最蠢的判断。"**

**Type 3 - 时间倒错**（暗示重生）：
- "我到达顶峰的那天，终于到了。"
- "我们曾经站在这里 之前一切被烧毁。"
- 移植：**"这一刻，我躺在出租车后座，怀里抱着活生生的念念。上一世的此刻，我正在签字。"**

**Type 4 - 设问/自问**（节奏切换）：
- "如果我们能再选择一次呢?"
- "她甚至不能让我拥有这个？"
- 移植：**"我做错了什么？不过是让女儿被自己的小婶看一眼。"**

## 🎬 场景级模板（结构性骨架）

### 开篇钩子（前 200 字必须包含）

**模板（仿 script_5431）**：
```
现在时态/动作镜头（5-15 字）
    ↓
角色一句"反常宣言"（≤15 字，揭示冲突大方向）
    ↓
反派一句"傲慢回应"或情境一句"价值反差"
    ↓
女主 OS 一句"复仇决心或痛悔"（≤25 字）
    ↓
闪回 / 场景切换标记
```

**应用到《蓉颜小厨》第 1 节**：
```
出租车颠簸了一下。
念念的小手揪紧我的衣角："妈妈，我不想去蓉蓉小婶家。"
我心里"咯噔"一声——上一世，我没听见这句话。
那一刻，念念才三岁，我抱着她，决定不再签那张该签的合同。
```

### 矛盾展开（200-600 字）

**结构**：
1. 一句"环境/动作"切换到下一个时空
2. 反派出场，用 1 句"道德绑架"或"阴阳怪气"
3. 女主一句沉默 + OS 一句"暗暗对照前世"
4. 女儿/孩子一句无意识童言（最催泪）
5. 反派变本加厉一句
6. 女主决定行动（OS + 动作）

### 打脸高潮（500 字+）

**节奏**：3 拍

| 拍 | 长度 | 内容 |
|---|---|---|
| 拍 1 - 设套 | 80-120 字 | 女主用一个看似"小事"的细节，反派毫无察觉 |
| 拍 2 - 反派自爆 | 60-100 字 | 反派以为占了便宜，得意忘形 |
| 拍 3 - 实锤+定论 | 40-80 字 | 女主一句话+一个动作，打回原形 |

每拍之间必须有"过渡 OS"（5-15 字）：
- "我等的就是这一刻。"
- "这才刚开始。"
- "你以为我是上辈子那个？"

## ✋ 黑名单（出现一次 = 整段废）

| 禁用句式 | 原因 | 改成 |
|---|---|---|
| "她默默发誓" | 戏剧腔 | 直接给 OS 句 |
| "心如刀绞" | 老套 | 给一个身体反应（"我手指攥进掌心，指甲嵌进肉里"） |
| "她不知道她将面对什么" | 上帝视角廉价 | 用 OS 自语 |
| "他冷冷一笑" | 影视描写 | 给具体动作 |
| "宝贝，妈妈一定会保护你" | 立 flag 假承诺 | 给具体行动 |
| "我们换一个家" | 霸总腔 | 用孩子能懂的话 |
| "买摄像头拍照" | 解释过度 | 用沉默动作 |
| "出租车司机问她" | 与剧情无关 | 删 |
| "她从来都是个善良的人" | 标签化 | 用事件展示 |

## 📐 自查清单（写完每一节必跑）

### A. 数值核查（机器可量化）
- [ ] 全文对白平均字数在 12-18 之间
- [ ] 长对白（>30字）占比 ≤ 10%
- [ ] 情绪括号只用白名单 8 种
- [ ] 没有连续 3 句以上 >25 字
- [ ] 开篇 200 字内有 OS 钩子

### B. 句式核查（人工对照黑名单）
- [ ] 不含 "默默"、"暗暗"、"心如" 这类抽象动作
- [ ] 不含解释性废话（"为什么呢？因为..."）
- [ ] 不含旁白对孩子说教（"宝贝你要..."）
- [ ] 反派对白属于 ABC 三类之一

### C. 人物核查
- [ ] 林禾的话有"林禾人物卡"里的特征（隐忍+冷峻）
- [ ] 念念的话像 3 岁小孩（不像 18 岁）
- [ ] 蓉蓉的话有阴阳怪气感
- [ ] 周文博的话有"自我感动"特征（仿 Charlie Barber / Vicomte）

### D. 节奏核查
- [ ] 每 300 字必有 1 个"反差/反转/钩子"
- [ ] 每 500 字必有 1 个对白爆点
- [ ] 没有连续 200 字纯描写无对白

## 🔧 使用方法

**写《蓉颜小厨》任何一节之前**：

1. 读这个指令包（约 5 分钟）
2. 读最相关的 1 个真实剧本片段（如 `素材/范文库/短剧资源/DramaBench中文剧本/script_5431_*.md`）
3. 读情节卡（`情节卡/节卡/第XX节-*.md`）和涉及的人物卡
4. 写
5. 写完跑自查清单
6. 不过关，重写
7. 过关后给用户
