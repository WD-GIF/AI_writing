## 🔍 三层检索法

### Layer 1：题材关键词检索（粗筛 → 50 个）

```python
# 例：第 1 节是"丧女母亲重生复仇"
themes = ['重生', '家庭背叛', '虐待孩子', '渣男出轨', '打脸虐渣', '食物伤害']
# 在 DramaBench 500 个剧本里匹配，按命中数排序
```

→ 已经做好：`_选中样本_15个.json`（针对《蓉颜小厨》整体题材）

### Layer 2：节级精细匹配（细筛 → 3-5 个）

每个节有不同侧重，要再筛：

| 节 | 关键题材 | 推荐样本 |
|---|---|---|
| 第 1 节 出租车后座 | 重生触发 + 母女私下沟通 | script_5431（开篇）/ script_3488（重生瞬间） |
| 第 2 节 发布会现场 | 婆家直面冲突 + 妯娌阴阳 | script_5431（嫂子道德绑架） / script_4481（妯娌斗争） |
| 第 3 节 直播翻车 | 公开打脸 + 反派自爆 | script_2927（公开复仇）/ script_5432（揭露真相） |
| 第 4 节 食物中毒 | 因果报应 + 法律介入 | script_2095（豪门审判）/ script_3773（毒杀复仇） |
| ... | ... | ... |

### Layer 3：句式定位（精读 → 1-2 个）

进入要写的节后，去对应剧本里找"目标句式"：

| 想写什么 | 去哪段找 |
|---|---|
| 女主第一句 OS 钩子 | script_5431 开篇 Qinqin 的"He should lose everything" |
| 反派道德绑架嫂子话 | script_5431 Sister-in-law 的台词全段 |
| 软弱丈夫摇摆话 | script_5431 Ayue 的"Ten minutes" |

## 🛠️ 检索脚本

### 主题分类检索

```python
#!/usr/bin/env python3
"""按题材关键词检索 DramaBench 500 个剧本"""
import json
from pathlib import Path

themes_kw = {
    '你要找的题材': ['关键词1', 'keyword2', '关键词3'],
}

records = []
with open("素材/范文库/短剧资源/DramaBench真实剧本_英文/dramabench_continuation_500.jsonl") as f:
    for line in f:
        if line.strip():
            records.append(json.loads(line))

def score(rec):
    text = (rec.get('title','') + ' ' + rec.get('description','') + 
            ' ' + rec.get('context','') + ' ' + rec.get('continuation','')).lower()
    s = sum(1 for kws in themes_kw.values() if any(kw in text for kw in kws))
    return s

scored = sorted(records, key=lambda r: -score(r))
for r in scored[:10]:
    print(f"{r['id']} 《{r.get('title','')}》: {r.get('description','')[:80]}")
```

### 句式样本检索

```python
"""在指定剧本里找含某关键词的台词"""
from pathlib import Path
import re

target_id = "5431"
keyword = "妈妈"

for f in Path("素材/范文库/短剧资源/DramaBench中文剧本").glob(f"script_{target_id}*.md"):
    txt = f.read_text(encoding='utf-8')
    for line in txt.split('\n'):
        if keyword in line and '**' not in line and 5 <= len(line.strip()) <= 30:
            print(line.strip())
```

## 🚧 注意事项

1. **不要全本读** — 一个剧本读 1-2 个场景即可，否则会被它的剧情污染
2. **学句式不学剧情** — 别把 script_5431 的剧情搬到《蓉颜小厨》
3. **混合 3 个样本** — 防止过拟合任何单一剧本
4. **真实剧本永远比 AI 自创的好** — 当卡壳时去检索，不要靠"创作"
