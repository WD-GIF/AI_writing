# 桥段卡 · 母女安抚场

> **桥段卡的作用**：把全文反复出现的"情节模式"标准化——这样每次写到都不会"重新发明轮子"，也不会"和上次写得一样"。
>
> 本桥段在**第 1 节、第 8 节、第 10 节**都会出现，每次都不一样。

---

## 一、桥段定义

**母女安抚场**：林禾对 5 岁念念的安抚戏。
- **不是**：林禾对外人发飙
- **不是**：林禾内心独白
- **是**：林禾**蹲下来，用动作或极少的话，让女儿感到安全**的戏

---

## 二、本桥段的核心法则（铁律）

| 法则 | 详解 |
|---|---|
| **动作 > 语言** | 林禾的爱用动作表达——抱、摸头、递东西、披外套 |
| **句子短到不能再短** | 林禾对女儿说的话 < 5 字 |
| **不立 flag** | 不说"妈妈以后再也不让 XX"——成年人的决定不需要孩子记住 |
| **不替孩子做决定** | 不说"妈妈给你换个家"——要用问句"你想 X 吗" |
| **不主动说"对不起"** | 林禾说不出口（人物卡设定） |
| **不抒情描写** | 不写"她的眼睛像星星"——专注当下 |

---

## 三、3 种不同强度的"母女安抚场"

> 不同节用不同强度。**安抚不是越深情越好，要看节奏需要**。

### 🟦 强度 1：第 1 节（首次重生 · 信息密集）

**目标**：让女儿说出 3 件具体被虐待事
**林禾的姿态**：蹲下 / 抱着 / 听
**语言强度**：低（问句多于陈述）

模板：
```
（动作：抱）
"念念。"
"嗯。"
"你认真说，妈妈听。"
"为什么不想去小婶家？"

[女儿一次说一句，慢慢说]
[妈妈用动作回应：摸头、握手、捂在胸口]

（结尾不立 flag）
"妈妈听见了。"
（不解释要做什么）
```

---

### 🟨 强度 2：第 8 节（婆婆崩裂后 · 余震）

**目标**：让女儿看到"大人世界结束了"的瞬间
**林禾的姿态**：站着 / 把女儿挡在身后 / 不解释
**语言强度**：中（短陈述句）

模板：
```
（婆婆在外哭闹）

念念：（攥小耳朵）"妈妈……"
林禾：（蹲下来，与女儿同高）"看着妈妈。"
念念：（看妈妈）
林禾：（一字一句）"妈妈在这里。"

（不解释发生了什么）
（不说"奶奶以后不会来了"——念念不需要这个承诺）
（直接关门）

（关上门后林禾才允许自己哭——但不让念念看）
```

---

### 🟩 强度 3：第 10 节（半年后 · 治愈）

**目标**：建立"新的日常"
**林禾的姿态**：随意 / 在做事 / 顺手回应
**语言强度**：极低（生活化短句）

模板：
```
"妈妈！"
"怎么了。"
"我吃饱了。"
"好。"
（接过碗去洗）
```

→ **结局的"安抚"不是煽情，是日常**。林禾终于学会**听见女儿叫她**——这就是疗愈。

## 四、❌ 本桥段绝对不要这么写

### A. 语言层面

1. ❌ "**今天之后，没有人能再让你蹲下来吃饭。也没有人能再叫你赔钱货。妈妈说的。**"
   → 对仗 + 立 flag + "妈妈说的"盖章——**三重戏剧腔灾区**
   
2. ❌ "**对不起念念，对不起，妈妈以为爸爸会照顾你**"
   → 主动道歉 + 甩锅给爸爸——**林禾不会这么说**

3. ❌ "**你怎么不早说啊？**"
   → 这是和 18 岁孩子的语气，不是和 5 岁的

4. ❌ "**妈妈给你换个家**"
   → 替孩子做决定 + "家"是抽象概念

5. ❌ "**一箱一箱地买苹果**"
   → 夸张数量 = PUA 式表演

### B. 动作层面

6. ❌ "**她把女儿紧紧抱在怀里，泪水模糊了视线**"
   → 自我抒情 + 让女儿看见妈妈哭——林禾不会让女儿看见她哭

7. ❌ "**林禾的眼神变得无比坚定**"
   → 给读者递信号——林禾不会给读者看"决心"

8. ❌ "**母亲发誓要让这些人付出代价**"
   → 内心戏外露——林禾的复仇是延迟反应，不在这里发誓

### C. 节奏层面

9. ❌ "**她们抱了很久很久**"
   → 时间模糊——爽文里时间必须具体

10. ❌ "**沉默良久后，她终于开口**"
    → "良久""终于"——AI 套话

---

## 五、✅ 本桥段必须做到

1. ✅ 林禾用至少 **1 个动作**（蹲下/抱/摸头/递东西/披外套）
2. ✅ 念念出现**至少 1 次小耳朵动作**（攥/抱/递/看）
3. ✅ 林禾说的话**全部 < 5 字**
4. ✅ 念念说的话**全部 < 8 字**
5. ✅ **不出现"对不起" "我以后" "妈妈说的"**
6. ✅ **结尾林禾没立 flag**——只有"妈妈在"或"好"

---

## 六、本桥段的"情绪温度"调节

| 节 | 情绪温度 | 表现 |
|---|---|---|
| **第 1 节** | 🔥 高（首次震荡） | 林禾蹲下、抱紧、深呼吸；念念怯怯说真话 |
| **第 8 节** | 🌡 中（震荡后稳定） | 林禾平静、念念已经能站直 |
| **第 10 节** | 💚 低（日常治愈） | 顺手接过碗，几乎没有"安抚"的戏剧感 |

→ **不要每次都用第 1 节那种强度**。安抚的"深度"应该**和情节同步降温**。

## 七、对照参考

### ❌ AI 默认会写的母女安抚场（**反面教材**）

```
林禾紧紧地把女儿抱在怀里，泪水如决堤的洪水般涌出。

"念念，对不起。妈妈对不起你。"

"是妈妈的错。是妈妈太忙了。"

"从今天开始，妈妈再也不让任何人欺负你。"

"妈妈会用全部的力量保护你。"

"宝贝，你想要什么妈妈都给你。要苹果？妈妈给你买一卡车。要 Hello Kitty？妈妈给你买整个店。"

念念在妈妈怀里抽泣着，小手紧紧抓住妈妈的衣服，仿佛害怕妈妈会消失。

"妈妈……妈妈我爱你……"

林禾的眼神变得无比坚定。

"念念，妈妈发誓，今天之后，没有人能让你不开心。"
```

**这段为什么是 AI 灾区**：
- "决堤的洪水"——比喻
- "对不起"开场——林禾说不出口
- 立 flag："再也不让任何人欺负"
- "整个店"——夸张数量
- "妈妈我爱你"——5 岁不会这么说
- "眼神变得无比坚定"——给读者看
- "今天之后……"——戏剧台词

### ✅ 真实的母女安抚场

```
我把她搂进怀里。

"念念。"
"嗯。"
"你认真说，妈妈听。"
"为什么不想去小婶家。"

念念抿着嘴。

"妈妈不生气。"
"妈妈听。"

"……爸爸和小婶只喜欢豆豆。"
"小婶说豆豆是周家的根。"
"我是别人家的。"

[沉默几秒]

念念又说，"过年时小婶把我那盘菜放在地上。"

我握住她小手。
```

**这段为什么对**：
- 全部短句
- 林禾每句 < 5 字
- 没有"对不起"
- 没有立 flag
- **用握手代替"我爱你"**
- 一次一句的节奏
