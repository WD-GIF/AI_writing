## 🛠️ 怎么用

### 用法 1：研究爆款剧情公式

每个简介都是"爆款骨架"，从中提炼**反复出现的桥段**：
- 重生类：第几集触发金手指？
- 追妻类：男主第几集开始追？
- 萌宝类：萌宝多大？哪集 DNA 实锤？

### 用法 2：找选题方向

按 tag 统计看哪个题材最火 → 直接选这类写。

### 用法 3：转化成小说

把短剧 1500-2500 字/集 × 50 集 = **75-125k 字网文**。
直接把简介当大纲，每集扩写成 1-3k 字 = 一篇完整网文。

## 🚨 如果还要剧本正文

dramamoyu 工具能拿到**视频播放链接**（mp4），不能直接拿剧本文字。

要从视频提取剧本：
1. 下载视频
2. 用 Whisper / Vosk 语音转文字（or OCR 烧制字幕）
3. 整理成剧本格式

工程量较大但**完全可行**。如果你想，我可以做这套工具。
