## 2. 黄金节奏数据（必背）

| 时间点 | 必须发生什么 | 不达标后果 |
|--------|--------------|-----------|
| 0-3 秒 | **冲突锚定**：捉奸 / 甩脸 / 耳光 / 砸 / 摔 / 撞 | 划走率高 |
| 30 秒 | **情绪爆破**：耳光 + 玻璃碎裂音效 + 配乐爆破 | 完播率 -172% |
| 每分钟 | **3-4 个情绪爆点** | 单调 |
| 第 7 分钟 | **第一个高潮位** | 推到第 5 分钟 → 留存 -23% |
| 第 8-12 集 | **一卡（一爆）**：付费短剧首付费点 | 付费转化扑街 |
| 第 26-30 集 | **二卡（二爆）**：第二付费点 | 中段流失 |
| 第 50-55 集 | **三卡（终极反转）** | 长尾完读率掉 |
| 每集结尾 | **cliffhanger 卡点** | 不点下一集 |

**核心引用**（来自 woshipm 6191708）：
> 创作一部爆款短剧，三大原则必不可少：画面刺激、剧情无脑、节奏爽快。
> 用户不为情怀买单，只为快感付费。

## 3. 主角设计黄金法则

**5 特写镜头人格锚定**：

woshipm 实测：第 1 集必须用 **5 个特写镜头**完成主角的人格 5 大锚点：

| 锚点 | 例（女主：苏曼，重生复仇剧）|
|------|----------------------------|
| 1. 核心标签（一句话能概括的人设）| 镜头 X：苏曼摸颈上红丝带（标志动作）|
| 2. 反差点 | 镜头 X：苏曼笑着递咖啡，转身眼神瞬间冷下来 |
| 3. 标志性能力 / 武器 | 镜头 X：苏曼一眼看穿对方手中的合同陷阱 |
| 4. 情感软肋 | 镜头 X：苏曼盯着童年照片走神 |
| 5. 决心或目标 | 镜头 X：苏曼撕掉合同，一字一顿：「这次，换我」|

5 个特写**必须在前 3 分钟内全部出现**，否则人物立不住。

## 5. 爽点公式（七宗罪驱动）

woshipm 实测，短剧爽点核心是 **7 宗罪映射**：

| 宗罪 | 短剧映射 | 经典爆款表现 |
|------|----------|------------|
| 贪婪 | **逆袭暴富 / 突然有钱** | 战神归来 / 龙王 |
| 色欲 | **性张力 / 暧昧拉扯** | 甜宠青梅 / 偏执霸总 |
| 虚荣 | **打脸炫富 / 当众羞辱前任** | 离婚追妻火葬场 |
| 暴怒 | **正义复仇 / 反向羞辱** | 重生复仇 / 老年觉醒 |
| 嫉妒 | **三角恋反派崩溃** | 绿茶白莲花打脸 |
| 暴食 | **沉浸物质爽感（豪车豪宅）**| 都市霸总 |
| 懒惰 | **响指系统不劳而获** | 男频脑洞超能力 |

每集至少 **1 个宗罪触发**。爆款剧每集会同时触发 2-3 个。

## 8. 平台审核红线（2026 年 1 月新规）

红果 / 番茄 / 抖音通用：

- **三级备案制**（广电总局 2025.2 通知 + 2026.1 落地）：< 100 万 / 100-300 万 / ≥ 300 万 三档分级
- **人工 + AI 双层审核**：历史观、演员权益、未成年保护是核心红线
- **AIGC 标识强制**：AI 短剧必须明示标识
- **下架数据**：2026.2 已下架未备案违规作品 4 万+
- **婚恋题材**：不得正当化出轨 / 小三 / 婚内强迫
- **复仇题材**：不得有具体犯罪手法教学（下毒细节、设套害人步骤）
- **老年题材**：不得过度描写"反子女虐待"，不得宣扬养生骗局，不得以老人之口涉医涉政
- **涉政、涉医（养生骗局）**：严禁
- **裸露 / 暴力 / 自杀**：分级审核
- **菩提临世下架案例**（参考）：8 亿播放被下架，问题集中在"宗教不当呈现"

## 9. 单集结构标准模板（180s × 60 grid）

按 woshipm 节奏数据 + 红果实测：

```
[0-3s]    冲突锚定 grid 1（特写）
[3-15s]   建立场景 grid 2-5
[15-30s]  第一爆破 grid 6-10（耳光 / 爆点）
[30-60s]  推进冲突 grid 11-20
[60-90s]  小高潮 grid 21-30
[90-120s] 反转 grid 31-40
[120-150s] 第二爆点 grid 41-50
[150-178s] 卡点铺陈 grid 51-59
[178-180s] 卡点 grid 60（cliffhanger）
```

每集 60 grid × 3s = 180s。每分钟 ~20 grid，每分钟必须 ≥ 3 个情绪爆点（即每 5-7 grid 1 个 cold_open / twist / satisfaction）。

## 10. 月度更新机制

- 第 6 节「现役爆款案例库」

4. 标注更新日期
