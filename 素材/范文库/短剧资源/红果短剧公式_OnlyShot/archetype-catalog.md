## 题材主原型（女频）

### 1. 重生复仇流
- **公式**：女主含冤死 → 重生 → 提前布局 → 一个个干掉前世仇人
- **核心爽点**：上帝视角碾压
- **集数典型**：60-100 集
- **代表**：《重生拒当冤大头》

### 2. 老年觉醒
- **公式**：被忽视的奶奶 → 一个事件触发 → 反向控制全家
- **核心爽点**：长辈智慧碾压晚辈算计
- **集数典型**：60-72 集
- **代表**：《奶奶觉醒后全家破产了》《十八岁太奶奶驾到》
- **状态（2026.5）**：🟡 红海前夜，需差异化

### 3. 神医 / 相师 / 命格
- **公式**：女主隐藏身份 → 救人 / 算命 → 揭露阴谋
- **核心爽点**：一眼看穿别人不知道的真相
- **集数典型**：50-80 集
- **代表**：《女相师》

### 4. 离婚追妻火葬场
- **公式**：男主嫌弃 → 离婚 → 女主开挂 → 男主追妻不得
- **核心爽点**：女主的"不要了"
- **代表**：《有本事别复婚》

### 5. 甜宠青梅
- **公式**：双向暗恋 → 误会 → 第三者插足 → 重新和好
- **核心爽点**：高浓度撒糖
- **代表**：《半糖青梅》

### 6. 婆媳虐恋
- **公式**：婆婆刁难 → 女主隐忍 → 反击 → 婆婆服软
- **核心爽点**：当众打脸婆婆
- **代表**：《家里家外 2》

### 7. 穿越当 XX
- **公式**：现代女主穿越 → 用现代知识降维 → 拿捏古代人
- **核心爽点**：信息差碾压
- **状态**：⚠️ 已饱和

### 8. 大女主商战
- **公式**：女主商业头脑 + 棋局 → 步步为营
- **核心爽点**：脑力打脸
- **代表**：《千金谋》

## 题材主原型（男频）

### 1. 都市脑洞 + 响指系统
- **公式**：男主获系统 → 响指打脸 → 越用越强
- **核心爽点**：响指 = 爽
- **代表**：番茄都市脑洞天花板

### 2. 都市修真种田
- **公式**：男主修真背景 → 都市低调 → 偶尔露一手
- **核心爽点**：扮猪吃虎

### 3. 战神归来 / 龙王
- **公式**：男主隐藏战神 → 被前妻轻视 → 身份暴露 → 全场跪
- **代表**：经典爽点

### 4. 赘婿逆袭
- **公式**：男主入赘 → 受白眼 → 真实身份是 X → 翻盘
- **状态**：⚠️ 已严重同质化

### 5. 悬疑民俗 / 风水
- **公式**：男主特殊职业 → 接案 → 民俗反转 → 揭秘
- **状态**：上升期

## 题材主原型（漫剧 / AI 漫剧）⭐ AI 优势所在

> **v0.1.2 注**：漫剧奇观流是 AI 真正的护城河（真人难拍 = AI 优势）。如果用户用 AI 出片，
> 这一类原型性价比最高。详见 `genre-flavors.md` C 章节。

### 1. 系统流
- **公式**：穿越 + 系统 → 完成任务越级打怪
- **代表**：《聚宝仙盆之杂灵根才是真 BOSS》
- **AI 优势场景**：系统面板浮空、任务进度条、积分爆增动效
- **金手指例**：响指系统、签到系统、合成系统、转生系统

### 2. 末世
- **公式**：丧尸 / 灾难 → 主角觉醒异能
- **代表**：《云浮梦若归》
- **AI 优势场景**：丧尸群、辐射云、烧毁城市、避难所内景
- **金手指例**：精神控制、空间储物、时间倒流、复活印记

### 3. 西游 / 神话（架空版）
- **公式**：神话人物现代化 / 异世界版本
- **代表**：《菩提临世真人 AI 版》（已下架，注意审核 — 改用架空神话体系）
- **AI 优势场景**：飞天、神兽、佛塔、仙界、九重天
- **审核警示**：不要用真实宗教人物（佛祖 / 菩萨 / 神仙 / 教派）→ 改架空

### 4. 修仙 / 玄幻
- **公式**：低武世界 → 修炼 → 飞升
- **代表**：番茄经典
- **AI 优势场景**：剑光破空、丹炉炼丹、洞府秘境、宗门大典、雷劫
- **金手指例**：上古传承、神兽契约、异灵根、神识共鸣

### 5. 沙雕漫
- **公式**：段子流 + AI 漫画
- **代表**：红果新增赛道
- **AI 优势场景**：夸张表情、变形动作、分镜爆炸（成本极低）

### 6. 异能觉醒（v0.1.2 新增）
- **公式**：普通人 → 觉醒异能 → 用异能打脸 → 越用越强
- **AI 优势场景**：瞳孔变色、长翅膀、空中漂浮、闪电掌、变身
- **金手指例**：火焰系、雷电系、空间系、精神系、治愈系

### 7. 异星 / 星际（v0.1.2 新增）
- **公式**：太空冒险 / 异星生存
- **AI 优势场景**：紫色天空、双月、悬浮岛、星舰、外星生物
- **少见但有差异化空间**

### 8. 古言玄幻（v0.1.2 新增）
- **公式**：女主穿越古代 + 玄学元素
- **AI 优势场景**：宫廷异变、灵符、占卜符号、命格图
- **代表**：《女相师》（玄学单元剧）— 介于精品悬疑流和漫剧奇观流之间

## 用法

阶段 1.2 引导用户：

1. 选 1 个题材主原型
2. 选 1 个副原型（可选）
3. 阶段 3 每个角色明确归类 1 个角色原型 + 1 个反差点

不要超原型——观众是来看熟悉的爽，不是看你创新。

但**反差点必须有**——没反差的纯原型 = 套路 = 观众觉得"又是这个"。

例：
- 隐藏战神 + 反差：怕老婆怕到当场跪
- 重生女王 + 反差：唯独对一只猫展示真情
- 老年觉醒 + 反差：会刷短视频、追星

## 反套路检查

注意 `references/cliche-detector.md` 的角色一级黑名单：

- ❌ 隐藏战神 / 神医 / 律师老人
- ❌ 配偶有不能说的秘密身世
- ❌ 主角是某高端职业的隐藏大佬

如果选了上述黑名单原型，必须在反差点里**明确反向**：
- 隐藏战神 → 让能力是真实习得（不是天降）+ 早早暴露
- 律师老人 → 让律师身份本身不再是反转点（一开场就告诉观众）
- 神医 → 让医术是网课学的 + 频频被质疑
