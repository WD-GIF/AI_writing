# 阶段 3：设定圣经深度细则（v0.1.1）

## 4 子步互动流程

### 3.1 角色清单（≤ 6 人）

```markdown
基于你的流派 + 题材，建议 6 个核心角色：

1. <女主>：<原型 + 反差点>
2. <男主或对手>：<>
3. <反派>：<>
4. <反派 2 / 帮凶>：<>
5. <配角 1>：<>
6. <终极幕后>（可选）：<>

OK？要换/砍/加？

注意：超 6 个就要砍（成本 + 观众记不住）。
```

### 3.2 每个角色的「5 特写镜头人格锚定」

woshipm 业界硬数据：主角需要 **5 个特写镜头**完成人格锚定。每个角色必须列出第 1 集里展示其性格的 5 个特写。

```markdown
# <女主>的 5 特写人格锚定

特写 1：<动作> — 展示「<核心标签>」
  例：苏砚秋摸颈上红丝带（标志动作 → 冷静）

特写 2：<动作> — 展示「<反差点>」
  例：苏砚秋笑着递咖啡，转身眼神瞬间冷下来（反差 → 表里不一）

特写 3：<动作> — 展示「<标志性能力>」
  例：苏砚秋一眼看穿对方手中的合同陷阱（能力 → 法律敏锐）

特写 4：<动作> — 展示「<情感软肋>」
  例：苏砚秋盯着童年照片走神（软肋 → 失去亲人）

特写 5：<动作> — 展示「<决心或目标>」
  例：苏砚秋撕掉合同：「这次，换我」（决心 → 复仇）

5 特写必须在前 3 分钟（即第 1 集前 60 grid 内）出现。
```

### 3.3 视觉母题 + 反差点 + 服装表 + 即梦 ref

每个角色完整档案：

```markdown
### 角色 N：<姓名>

- **原型**：<从 archetype-catalog.md 选>
- **反差点**：<一句话，必须有>
- **基本信息**：年龄 / 性别 / 身高 / 体型 / 职业
- **背景故事**（≤ 100 字）：<>
- **角色弧光**（在 72 集如何变化）：
  - 起点（第 1 集）：<状态>
  - 拐点 1（第 X 集）：<事件 → 状态变化>
  - 拐点 2（第 Y 集）：<>
  - 拐点 3（第 Z 集）：<>
  - 终点（第 72 集）：<状态>
- **5 特写人格锚定**（来自 3.2）
- **视觉母题**：≥ 2 个（反复出现的色彩 / 物件 / 动作 / 配饰）
- **象征意义**：<这个角色代表的主题>
- **标志性动作**（≥ 2 个）：<>
- **口头禅 / 经典台词**（≥ 3 句）：<>
- **服装表**（按阶段）：
  - 第 1-9 集：<>
  - 第 10-30 集：<>
  - 第 31-50 集：<>
  - 第 51-72 集：<>
  - 关键场合：<>
- **即梦角色 ref 提示词**（中文 + 英文）
- **角色关系**：<和其他角色的关系起点 / 终点>
```

### 3.4 写 3 份圣经

`05_角色圣经.md` + `06_场景圣经.md` + `07_道具圣经.md`

## 视觉母题示例

- 苏砚秋（律师奶奶）：**银杏叶胸针**（亡女遗物，每集都戴）+ **黑檀木拐杖**（藏证据）+ **手腕旧伤痕**（30 年前救婴儿留下）
- 林策（霸总）：**银色打火机**（焦虑反复打开关闭）
- 张兰（反派婆婆）：**枣红旗袍 + 满手金镯**（每次出场金镯哗啦响一下，**视觉母题 + 听觉母题**双重）

母题作用：
- 观众无意识中识别角色
- 镜头剪辑有重复元素
- 即梦多段视频生成时锚定角色一致性

## 场景 bible

只收录出现 ≥ 5 集的场景（72 集背景下，5 集是合理阈值）。每个场景：

```markdown
### 场景 N：<场景名>

- **场景 ID**：scene_<编号>
- **场景描述**（≤ 100 字）：物理空间、装饰、氛围、灯光
- **出现集数**：EP01, EP05, EP10, ...
- **情感色彩**（这个场景反复出现承载什么情绪）：<>
- **关键视觉元素**：<>
- **重复母题**：<场景中反复出现的元素>
- **即梦场景 ref 提示词**（中文）：用于生成场景四宫格 ref（4 个角度：全景 / 入口 / 关键陈设特写 / 窗外或对位视角）
```

---

## 道具 bible

只收录有剧情推动 / 象征意义的。日常物件不收录。

```markdown
### 道具 N：<道具名>

- **道具 ID**：prop_<编号>
- **描述**（≤ 50 字）：外观、材质、尺寸、颜色
- **出现集数**：<>
- **剧情功能**：<例：解开身世之谜的钥匙>
- **象征意义**：<例：象征女主对母亲的执念>
- **携带 / 使用规则**：<>
- **即梦道具 ref 提示词**（中文）：用于生成三视图（正面 / 侧面 / 背面 / 细节特写）
```

## 数量上限

| 项 | 上限 | 理由 |
|----|------|------|
| 主要角色 | 4 | 72 集容不下更多脸 |
| 次要角色 | 2 | 反派 + 工具人 |
| 场景 | 5 | 每多一个就多一份成本 |
| 道具 | 5 | 多了观众记不住 |

超上限要砍。砍的原则：合并 → 删除 → 替换为文字交代。

## 反套路检查（v0.1.1 新增）

阶段 3 完成前自检 `references/cliche-detector.md` 黑名单：

- 角色一级黑名单（隐藏战神 / 律师老人 / 神医）→ 必须在反差点里反向
- 角色二级黑名单（绿茶白莲花 / 婆婆刁难 / 三角恋第三者）→ 慎用
- 角色三级黑名单（人物名字模板「沈/苏/陆 + X玥/曼/砚/锦」）→ 换更具体的
