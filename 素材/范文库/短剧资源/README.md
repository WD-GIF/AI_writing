**为什么是英文?**

**用法**:
- 学**短剧的 Fountain 格式**（行业标准）
- 看真实剧本的**对白 / 舞台指示 / 镜头切换**怎么写
- 看续写、节奏切分（split_type=scene_boundary）的规律
- 中英对照可以训练你的剧本翻译能力

## 💡 完整的"短剧学习"工作流

### 第一步：用元数据找选题

打开 `红果短剧分类索引.md`，看 1524 部爆款的**题材分布**：

```
女频 Top 5 题材标签：
- 都市情感
- 重生
- 追妻火葬场
- 萌宝
- 打脸虐渣

男频 Top 5：
- 战神归来
- 都市
- 系统
- 重生
- 玄幻
```

**用法**：你想做的题材如果在 Top 10 里，就证明市场验证过；如果不在，要慎重。

### 第二步：用公式学结构

读 `红果短剧公式_OnlyShot/`：
- `opening-rules.md` → 学开篇 30 秒怎么钩人
- `rhythm-curve.md` → 学 80 集怎么分节奏（一卡/二卡/终极反转）
- `hook-design.md` → 学每集末钩子怎么写
- `satisfaction-matrix.md` → 学爽点密度怎么算

### 第三步：用英文剧本看真实结构

打开 `DramaBench真实剧本_英文/单文件/` 里任意一个：
- 看场景指示（INT. BEDROOM - DAY）
- 看人物上场（CHARACTER 出场行）
- 看对白格式
- 看节奏切分

虽然是英文，但**结构是中国短剧 100% 移植过来的**——拿来对比你的中文写作。

### 第四步：自己写

用：
- 选题（来自元数据）
- 结构（来自公式）
- 格式（来自英文剧本）
- 中文（你自己）

= 你的红果短剧本子。

## 🚨 真正"获取剧本正文"的合法/灰色路径

1. **注册 creator.shortdramas.com**（红果创作者中心）
   - 投稿审核后能拿到 IP 改编授权
   - 这是唯一**合法**且能拿到完整剧本协作流程的路径
   
2. **找 08i8 / 网盘资源社等付费站**
   - 单部剧本 500-2000 元
   - 灰色地带，不保证合规

3. **加编剧群 / 知识星球**
   - 行业群里有 share，但通常自己人才能进
   - 通过认识编剧入坑
   
4. **扒抖音 / 红果上的短剧字幕**（半完整）
   - 用 OCR / 视频字幕提取工具
   - 得到的是台词流水，**没有舞台指示、人物动作描述**
   - 这部分**我可以用爬虫帮你做**——如果你需要的话
