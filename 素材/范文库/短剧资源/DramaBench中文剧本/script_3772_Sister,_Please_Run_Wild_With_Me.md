# 《姐姐,请跟我一起狂野吧》

> **English Title**: Sister, Please Run Wild With Me

> **剧情简介**: 在旅馆里一个充满激情的夜晚之后,一个穿着婚纱的女人给男人留了钱,声称这只是为了取乐。 该男子被透露为一位强大的首席执行官,多年来一直无法忘记她,决定通过承认学生身份来追逐她。

## 角色表

- **HuoYanchuan**（HUO YANCHUAN）
- **NanWan**（NAN WAN）

---


### 🎬 内景．酒店客房 - 夜


HuoYanchuan刚刚洗完澡,他的下身上裹着一条白毛巾,从冰霜玻璃门后面出来。他的体形很强壮,肌肉精细。

NanWan, 穿着一件盛大的婚纱,坐在乱七八糟的床上,拿着她的手机, 仔细地看着他。

HuoYanchuan一步一步地走向床上,他看着NanWan,眼睛充满了侵略。

**NanWan**
自巴塞罗那以来第一次吗?

**HuoYanchuan**
这重要吗?

**NanWan**
确实 我不分享

**HuoYanchuan**
是吗 那为什么穿这裙子

NanWan拿起酒杯,喝一小口红酒。

**NanWan**
你能处理婚纱吗 还是姐姐需要教你?

**HuoYanchuan**
_（笑笑声）_
姐姐吗?

**NanWan**
我教你怎么做到的

HuoYanchuan倾斜靠近NanWan, 一只手踩在床上,另一只抚摸她的脸颊。

**HuoYanchuan**
你明天不会走路了

HuoYanchuan亲吻NanWan 两吻激情地掉到床上 房间的灯光暗淡

_淡入到：_



### 🎬 内景．酒店客房 - 日


第二天早上 碎掉的酒杯和衣服散落在地上

NanWan在床上变换成丝绸睡衣,她轻轻地摸着睡在她旁边的HuoYanchuan胸部,她的表情很复杂。

**NanWan**
_（自言自语）_
这是个错误,不能再发生了

NanWan下床 换成黑色的黑衣服 穿上高跟鞋 准备离开

HuoYanchuan突然醒来,从后面抓起她的手。NanWan转过身来,两只锁着眼睛。

NanWan 解脱,从她的手提包中拿出一堆现金,然后扔到HuoYanchuan上。

**NanWan**
为了房间 我们扯平了

**HuoYanchuan**
_（坐在床上,用毯子遮盖自己）_
你以为钱能解决这个?

**NanWan**
戒指 我订婚了 这是告别

**HuoYanchuan**
你跑开了

**NanWan**
_（紧靠着他,用她的手指抬起下巴）_
我们在一起总是很好,但有些事情没有改变

NanWan 直立,从她手中取下钻石戒指。

**NanWan**
我的订婚戒指 从我接受戒指那天起 我就一直想忘记你

NanWan把戒指放在HuoYanchuan前面,转身优雅地离开。在门前,她转过身去,给他一个吻,然后关上门,离开。

HuoYanchuan看着她的离开,他的表情从休克转向决心。他从钱包中拿出了一张照片,在海滩酒吧的一间小的NanWan。

HuoYanchuan (V.O.)
三年,南环,你以为我会放你走吗?

HuoYanchuan接起他的电话打电话

**HuoYanchuan**
取消东京协议,在南城大学商学院给我学生身份

警卫 (O. S.)
是,霍先生

HuoYanchuan挂起电话,捡起NanWan留下的戒指,眼睛变得坚定和占有。

HuoYanchuan (V.O.)
你想跑吗?

_淡出。_
