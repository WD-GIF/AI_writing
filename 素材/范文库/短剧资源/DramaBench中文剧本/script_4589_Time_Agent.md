# 《时 间代理》

> **English Title**: Time Agent

> **剧情简介**: 一位身患绝症的男人在被女友抛弃后试图自杀,但一位神秘的时代老人拯救了他,并给了他一张可以延长生命、看到死亡倒计时和操纵时间的袖手表。

## 角色表

- **李阳**（LI YANG）
- **TimeElder**（TIME ELDER）

---

国家(V.O.)
_（女朋友的声音）_
李杨,我们分手吧 我不会跟有绝症的人在一起


### 🎬 外景．艰苦的街道 - 夜


李阳, 身穿格子衬衣,在雨中缠住,他盖住嘴,咳嗽,手腕上有医疗手镯。


### 🎬 外景．屋顶 - 夜


李阳站在屋顶边缘,嘴角有血迹,眼睛空洞。下面是人群聚集,电话响起记录。

李阳向下看人群,

国家(V.O.)
_（女朋友的声音）_
好好看看你自己 看看你的真面目

李阳 (V.O.)
我给了你我拥有的一切,我是真的瞎了眼

李阳张开手臂 向后退 从高楼直下


### 🎬 外景．街道 - 夜


时间突然停了,街道上所有东西都冻僵了,男人手上的花瓣挂在空中,女孩手上的爆米花也停在空中。

白头发的白发TimeElder 慢慢走向坠落的李阳。

TimeElder伸展他的手。李阳的身体浮在空中,然后轻轻地放在地上。

李阳躺在地上,痛苦地呻吟着

**李阳**
啊!

李阳的摇晃醒了,发现自己安全地躺在地上。 心神不宁,他站起来看着周围的人群。

**李阳**
这是怎么回事?

TimeElder走过来找他 拿着一只手表

**TimeElder**
年轻人,既然你付钱买的 这支袖手表是你的了

**李阳**
袖口表?


### 🎬 外景．街角 - NIGHT (FLASHBACK)


TimeElder 停止了一个令人心烦的 李阳。

**TimeElder**
年轻人 想买点什么吗 这个袖手表不错 只有五百元

李阳给予苦笑,从他口袋里拿出所有现金,交给TimeElder。

**李阳**
老头,这钱对我来说已经毫无意义了 我不需要手表,你留着吧

李阳转身离开。


### 🎬 外景．街道 - 夜


TimeElder 将手表挂起来,它轻轻地发光。

**TimeElder**
它能衡量你们所剩的。拯救生命,增加半天,其余的就显现出来。

TimeElder将口表放在李阳的手中,然后消失。

李阳 看着他手上的手表,手表发光并形成手臂倒计时: 06: 59: 59 。

李阳盯着他的手臂上的数字,震惊地盯着他,然后看着周围的人群。

**李阳**
老头!

李阳的前女友出现在人群中,

待续
