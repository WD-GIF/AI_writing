# 《凤凰返回巢》

> **English Title**: Phoenix Returns to Nest

> **剧情简介**: 渔夫的女儿在救溺水的首席执行官之后 被带走了 在她生日那天把她和母亲分开

## 角色表

- **BodyguardA**（BODYGUARD A）
- **BodyguardB**（BODYGUARD B）
- **顾Peisheng**（GU PEISHENG）
- **徐Xiu**（XU XIU）
- **赵Xiaoxi**（ZHAO XIAOXI）

### 🎬 内景．海滩 - 黄昏


太阳在沙滩上落下 小浪对岸低声说

小女孩赵Xiaoxi在湿沙中蹲下,把光滑的石块堆到一个小心的塔里。

一架直升机转子的远端THUMP在静悄悄地切开。

她的母亲,渔夫徐Xiu, 划船到岸边。

**徐Xiu**
小西,小西! 小西!

小西抬头站着微笑

**赵Xiaoxi**
妈妈!

她向水边跑去

徐秀下船 抓着一串新捕到的鱼

**徐Xiu**
今天好运。

**赵Xiaoxi**
妈妈,你和爸爸很忙 今天是我生日,你能多和我待一会吗?

**徐Xiu**
_（软化）_
最后一季的最后一季要花你的艺术课的钱 然后我整晚都是你的了,好吗?

**赵Xiaoxi**
_（兴奋）_
成交!

**徐Xiu**
我们回家吧

他们握手,走在海滩上

直升机Engine ScREECHES在上空,一个SHADOW通行证。

然后是一次爆炸。直升机飞入海中, 发出巨大的水流。

渔民冻结。

在酸水上,一个叫顾Peisheng的男人 挣扎,喘气

**顾Peisheng**
救命... 救命... 救命...

徐秀时装,释放小西的手

**赵Xiaoxi**
_（惊恐）_
妈妈!

**徐Xiu**
_（实数）_
在这儿等着

**顾Peisheng**
_（下沉）_
我不能... 坚持住...

徐秀抓到他,抓住他的项圈

**徐Xiu**
小西 生命戒指

小西冲上船 解开生命戒指 扔给她妈妈

许秀把戒指推到Gu Peisheng手里

他的手指扣住它,就瘸了。他跌倒了,把戒指拉下来。

他的手被割掉,他消失了

小西看见妈妈也下沉了

她毫不犹豫地潜入

几辆黑色的SUV车加快了速度,停在海滩上。

**BodyguardA**

**BodyguardB**

**BodyguardB**
