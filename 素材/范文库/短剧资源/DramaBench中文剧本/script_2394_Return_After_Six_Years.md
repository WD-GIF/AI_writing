# 《六年后返回》

> **English Title**: Return After Six Years

> **剧情简介**: 一名妇女与儿子在一起六年后返回,为过去的背叛寻求正义,并准备收回从她手中夺走的东西。

## 角色表

- **Doctor**（DOCTOR）
- **林Qianqian**（LIN QIANQIAN）
- **温成**（WEN CHENG）
- **温Ruan**（WEN RUAN）
- **谢静Hang**（XIE JING HANG）

---


### 🎬 内景．机场临时机场 - 日


温Ruan(Wen家庭继承人)身穿黑色商业西装,戴墨镜,拉起行李箱,手握着她儿子温成 (Wen Ruan的儿子)的手,当时她带着强大的力量行走。

温Ruan摘除她的太阳镜, 她的眼睛定下来。

**温Ruan**
六年了,该是结账的时候了

**温成**
_（看着他的母亲）_
妈妈,爸爸会在这里吗?

**温Ruan**
_（她的双眼暗暗）_
你父亲六年前去世了 我们是来伸张正义的


### 🎬 内景．医院的 - DAY (FLASHBACK)


16年前 6年

温Ruan躺在医院的床上,看起来很虚弱。 林Qianqian(Lin家族私生女儿)带着一个婴儿穿着衣着,带着恶毒的笑容站在床边。

**林Qianqian**
那孩子没来,静汉哥问候你

**温Ruan**
_（疼痛）_
不... 他不会...

**林Qianqian**
他失明的时候你很舒适 现在?

林Qianqian 转向附近的Doctor。

**林Qianqian**
她不能活着离开这个房间 转岗一小时后就没事了

**Doctor**
_（犹豫）_
议定书 --

**林Qianqian**

会得到赔偿的 十倍

林Qianqian 出口。 Doctor 用注射器接近。

**温Ruan**
_（穿过坚固的牙齿）_
等等,我出价会翻三倍
_（紧紧紧系紧紧的锁链）_
我是文家失传的继承人 证据在我项链里

Doctor暂停, 检查雕刻:“ Wen Estate - First Paughter.” 他的表达方式变了。

**Doctor**
_（辞职）_
推,现在

温Ruan抓住床单 痛苦地推着 她下面沾满血迹

**温Ruan**
_（尖叫）_
啊哈! Aaghh!

她崩溃了


### 🎬 内景．医院冷藏室 - 日


林Qianqian现在穿着与温Ruan相同的医院礼服,

**林Qianqian**
_（快乐快乐）_
晶杭,手术成功了 我们的儿子在等

**谢静Hang**
_（斜视,偏邪,）_
孟明,你的声音听起来...

**林Qianqian**
_（倾靠近）_
一切都变了 但这一刻...

**谢静Hang**
_（抚摸婴儿的脸）_
我们的孩子...

谢静Hang摇篮婴儿, 他的表情温柔。

温Ruan从阴影中抱着她的婴儿表,把毁灭刻在她的特征上,她紧紧地抓住孩子,眼神从不离开扭曲的场景,眼泪开始滴落。
