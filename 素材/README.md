# 素材区

把你想让 AI 参考的所有原始资料放到这个文件夹里。

## 可以放什么

- 📚 **参考小说片段**：风格参考、桥段参考（注意版权，仅作内部参考）
- 🎭 **人物原型**：现实/影视人物的描述，作为角色塑造基础
- 🌍 **世界观素材**：神话传说、历史资料、设定参考
- 💡 **灵感碎片**：随手记下的句子、点子、画面
- 📝 **过往草稿**：你自己写过的片段
- 🖼️ **图片/截图**：场景图、人物图（图片放进来即可，AI 能读取文件名和文字说明）

## 命名建议

```
素材/
├── 参考-某某小说-修仙体系.md
├── 人物原型-冷酷反派.md
├── 灵感-2026年6月.md
└── 图片/
    └── 场景-雪山秘境.png
```

文件名清晰一点，AI 检索起来更准。
