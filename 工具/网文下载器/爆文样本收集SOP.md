# 爆文样本收集 SOP（标准操作流程）

> 给"完全不会用工具"的你一份**手把手操作清单**。
> 目标：在 1 天内收集 30-50 篇高质量爆文样本，喂给 AI 提取爆点公式。

---

## 阶段 1 · 准备工具（一次性，30 分钟）

### ☑ 1.1 装 Python（如已装跳过）

- 打开 [python.org](https://www.python.org/downloads/)
- 下载最新版 3.x
- **Windows 安装时勾选 "Add Python to PATH"**（不勾选后面会很麻烦）
- 装完打开"命令提示符"（Win+R 输入 cmd），输入 `python --version`，看到版本号就成

### ☑ 1.2 装依赖（30 秒）

把这个仓库克隆/下载到本地，进入 `工具/网文下载器/` 文件夹，命令行运行：

```bash
pip install -r requirements.txt
```

完成后输入 `pip list` 能看到 `requests`、`beautifulsoup4` 等几个包。

---

## 阶段 2 · 锁定题材方向（10 分钟）

### ☑ 2.1 选 3 个核心题材（不要贪多）

参考 `旧爆点/00-总览.md`，建议这 3 个：

1. **追妻火葬场**（女频热度榜第二）
2. **重生复仇**（基本盘最大）
3. **萌宝认亲 + 隐婚** 或 **打脸虐渣**（看你偏好）

### ☑ 2.2 给每个题材建一个收集文件夹

```
素材/范文库/
├── 追妻火葬场/
├── 重生复仇/
└── 打脸虐渣/
```

---

## 阶段 3 · 收集 URL（半天）

### ☑ 3.1 公众号短篇爽文（首选 · 字数最匹配）

#### 推荐关注的号

| 公众号 | 风格 |
|---|---|
| 夜读 | 治愈系 + 重生 |
| 清晨 | 励志逆袭 |
| 深夜读物 | 重生复仇 |
| 有故事的人 | 婆媳/婚恋 |
| 故事会 | 经典爽文 |
| 樊登读书 | 偶尔有改编爽文 |

#### 操作步骤

1. **微信里搜公众号** → 关注
2. **查看历史推送** → 挑标题里有"重生""追妻""打脸"关键词的
3. **看阅读量** → **10万+** 优先（爆款的硬指标）
4. **点开文章** → 右上角 "..." → **"复制链接"**
5. **粘贴到 `urls.txt`**

每个题材收 **10-15 篇**就够了。

### ☑ 3.2 抖音推文（曲线救国）

1. 抖音搜 `重生爽文` / `追妻火葬场推文`
2. 看视频找原文链接（评论区或主页置顶）
3. 大部分是公众号链接，可以用上面方法收

### ☑ 3.3 笔趣阁系列（适合长篇分析）

如果你**也想看长篇怎么写**：

1. 浏览器搜 "笔趣阁"，选一个能打开的镜像站
2. 进入"短篇专区"或"完结榜"
3. 复制小说**目录页 URL**（不是单章 URL）
4. 粘贴到 `urls.txt`

⚠️ 笔趣阁系列**全是盗版**——只能内部学习，不要外传。

### ☑ 3.4 知乎盐选（高质量但需要会员）

1. 知乎搜 `重生故事`、`完美陷阱`、`我和` 这类关键词
2. 找盐选会员故事
3. 链接放进 `urls.txt`

知乎专栏文章可以直接下载，盐选会员文章需要登录会有问题，目前工具不支持，**手动复制粘贴更快**。

---

## 阶段 4 · 批量下载（30 分钟）

### ☑ 4.1 准备 URL 列表

按题材建 3 个 txt 文件：

```
工具/网文下载器/
├── urls_追妻.txt
├── urls_重生.txt
└── urls_打脸.txt
```

每个文件一行一个 URL：

```
# 追妻火葬场题材爆款，2026 年 6 月收集
https://mp.weixin.qq.com/s/aaaa
https://mp.weixin.qq.com/s/bbbb
https://mp.weixin.qq.com/s/cccc
...
```

### ☑ 4.2 命令行批量下载

进入 `工具/网文下载器/` 目录，按题材分别跑：

```bash
# 追妻火葬场
python novel_downloader.py --batch urls_追妻.txt --min 5000 --max 15000 --out ../../素材/范文库/追妻火葬场

# 重生复仇
python novel_downloader.py --batch urls_重生.txt --min 5000 --max 15000 --out ../../素材/范文库/重生复仇

# 打脸虐渣
python novel_downloader.py --batch urls_打脸.txt --min 5000 --max 15000 --out ../../素材/范文库/打脸虐渣
```

字数筛选范围用 **5000-15000**（你目标是 8-12k，但给点缓冲）。

### ☑ 4.3 等待下载完成

每个题材 10-15 篇，大约 5-10 分钟。
有进度条，去喝杯水。

---

## 阶段 5 · 人工筛选（关键 · 1 小时）

工具不知道哪篇是真的好。**你自己得过一遍**。

### ☑ 5.1 进入下载目录

```
素材/范文库/追妻火葬场/
├── 文章1.md
├── 文章2.md
├── ...
└── _筛除/                     ← 字数不符的，可以丢
```

### ☑ 5.2 逐篇快读（每篇 3-5 分钟）

3 个判断标准：

| 判断点 | 看什么 |
|---|---|
| **开头** | 前 300 字能不能立刻抓住你 |
| **爆点** | 反派被怼时你有没有想拍腿 |
| **结尾** | 钩子结得好不好 |

### ☑ 5.3 给每篇打标签（用文件名）

把好的改名加 ⭐ 标记：

```
⭐⭐⭐-重生我女儿被害死了.md      ← 必读级
⭐⭐-公婆抢家产被反杀.md          ← 不错
⭐-平庸版重生.md                  ← 一般
```

不好的直接删，或者移到 `素材/范文库/_弃用/`。

### ☑ 5.4 每个题材最终留 **5-8 篇⭐⭐⭐**

少而精。AI 学 5 篇好的，比学 50 篇普通的强。

---

## 阶段 6 · 给 AI 看 + 提取公式（自动）

### ☑ 6.1 把筛选好的文件**整理到仓库**

```
素材/范文库/
├── 追妻火葬场/
│   ├── ⭐⭐⭐-范文1.md
│   ├── ⭐⭐⭐-范文2.md
│   └── ...
├── 重生复仇/
│   └── ...
└── 打脸虐渣/
    └── ...
```

### ☑ 6.2 告诉 AI

下次对话直接说：

> "我把 X 题材的 Y 篇范文上传到 `素材/范文库/X/`，帮我提取每篇的爆点公式，更新 `旧爆点/` 数据库。"

AI 会自动：
- 提取每篇的开头钩子
- 提取每篇的爆点节奏
- 提取每篇的反派塌房模板
- 提取每篇的章末钩子
- 整理成可复用的桥段卡

---

## 阶段 7 · 持续维护

每个月做一次：

- 关注 1-2 个新的公众号
- 收 10-20 篇新爆文
- 跑一次工具
- 筛选 5-8 篇精品
- 替换掉过时的旧范文

**保持你的"爆文金矿"鲜活**。

---

## 🚨 常见坑

### 坑 1：贪多

下了 200 篇看不完，等于没下。**30-50 篇够你用 3 个月**。

### 坑 2：不筛选

直接把 50 篇全喂给 AI，AI 会被平均化干扰。**精选 > 数量**。

### 坑 3：只看公众号

公众号风格偏文艺。你的目标是**下沉爽文**，应该公众号 + 笔趣阁 + 知乎**混合 1:1:1**。

### 坑 4：所有题材一起做

3 个题材一起做，**风格会被串味**。建议**一次只 focus 1 个题材**，做完再下一个。

### 坑 5：不做标注

下完不写"为什么这篇是爆款"，过两周自己都忘了为啥留下它。**留下的每篇都该有一句标签**：

```
⭐⭐⭐-XXX.md
[标签: 开篇钩子+反派群嘲+丈夫出轨实锤]
[爆点: 用孩子的尸体让丈夫认错]
[长度: 11200 字]
[结构: 1 节 9 小段]
```
