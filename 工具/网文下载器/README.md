# 爆文批量下载工具

> 用于批量收集网文爽文素材，喂给 AI 做风格学习和爆点提取。
> **仅供个人学习研究使用**。

---

## 📦 安装（一次性，5 分钟）

### 1. 安装 Python（如果还没有）

- **Windows**：去 [python.org](https://www.python.org/downloads/) 下载安装，**安装时勾选 "Add Python to PATH"**
- **Mac**：自带 Python 3，命令行运行 `python3 --version` 确认
- **检查是否成功**：打开命令行，输入 `python --version`，能看到 3.x 就行

### 2. 安装依赖

打开命令行（终端），进入这个文件夹，运行：

```bash
pip install -r requirements.txt
```

如果 pip 提示找不到，用 `pip3` 代替 `pip`。

完成后输入 `pip list`，能看到 `requests`、`beautifulsoup4`、`html2text`、`tqdm` 就说明装好了。

## 🚀 使用方法

### 模式 1：下载单本笔趣阁小说（全本）

```bash
python novel_downloader.py --url "https://www.xbiquge.so/book/12345/"
```

会自动：
- 抓取目录页
- 逐章下载（带进度条）
- 章节间随机延迟（防被封）
- 合并成一个 `.txt` 文件，保存到 `素材/范文库/`

### 模式 2：下载单篇公众号文章

```bash
python novel_downloader.py --url "https://mp.weixin.qq.com/s/xxxxx"
```

会输出一个**带格式的 Markdown 文件**到 `素材/范文库/`，包含：
- 标题、作者、字数、链接
- 正文（自动去广告、去图片占位符）

### 模式 3：批量下载（最实用）

**步骤 1**：把所有要下的 URL 整理到 `urls.txt`，**一行一个**：

```
# 这是注释，会被忽略
https://mp.weixin.qq.com/s/aaaa
https://mp.weixin.qq.com/s/bbbb
https://www.xbiquge.so/book/11111/
https://www.xbiquge.so/book/22222/
```

**步骤 2**：运行批量命令：

```bash
python novel_downloader.py --batch urls.txt
```

工具会：
- 自动检测每个 URL 的站点类型
- 逐个下载
- URL 间随机延迟 2-5 秒
- 失败的自动跳过，**不会因为一个挂掉中断全部**

### 模式 4：批量下载 + **按字数筛选**（推荐）

```bash
python novel_downloader.py --batch urls.txt --min 8000 --max 12000
```

- 只保留 **8000-12000 字**（你目标的短篇爽文范围）
- 字数不在范围的会被移到 `素材/范文库/_筛除/` 文件夹（不删除，可以回头看）

### 模式 5：自定义输出目录

```bash
python novel_downloader.py --batch urls.txt --out 我的范文库/追妻火葬场
```

按题材分目录，方便后续训练 AI。

## 📂 输出结构

默认输出到 `素材/范文库/` 下：

```
素材/范文库/
├── 重生后我才发现我女儿是被害死的.md          ← 公众号文章
├── 我说他在偷情他却让我承担一切.md            ← 公众号文章
├── 某某重生爽文.txt                          ← 笔趣阁全本
└── _筛除/                                    ← 字数不符的
    └── 太短或太长的.txt
```

## ⚙️ 命令速查

| 参数 | 作用 |
|---|---|
| `--url <URL>` | 下载单个 URL |
| `--batch <file>` | 批量下载，从文件读 URL 列表 |
| `--type biquge\|wechat\|generic` | 强制指定站点类型（一般不用） |
| `--min <n>` | 最小中文字数 |
| `--max <n>` | 最大中文字数 |
| `--out <dir>` | 输出目录 |

## 🎯 推荐用法（针对你想做的事）

### 收集 8000-12000 字短篇爽文

1. 关注几个公众号短篇号（夜读 / 清晨 / 深夜读物 / 有故事的人 等）
2. 找历史推送，挑爆款（10万+、点赞高）的，把 URL 复制到 `urls.txt`
3. 一次性收 30-50 篇，然后：

```bash
python novel_downloader.py --batch urls.txt --min 8000 --max 12000 --out 素材/范文库/2026-6月
```

4. 进入输出目录，**人工浏览一遍**，把不符合的删掉
5. 留下来的 10-20 篇就是你的**金料**了

## ⚠️ 注意事项

### 法律 & 道德

- ✅ **个人学习/研究**：完全合法
- ✅ **提取风格/公式给 AI 训练**：内部使用，没问题
- ❌ **二次发布**：抄袭、侵权
- ❌ **删除版权信息冒充原创**：违法
- ❌ **批量爬取 VIP 付费章节**：违反平台条款
- ❌ **高频率攻击式爬取**：可能触发反爬法律风险

工具默认有 **1-2.5 秒章节延迟 + 2-5 秒 URL 间延迟**，已经够友好。**不要去掉延迟**。

### 反爬应对

如果遇到 403 / 验证码：
- 等 30 分钟再试
- 换个 Wi-Fi（家用 / 手机热点）
- 增大延迟（修改 `DELAY_RANGE`）
- 笔趣阁系列偶尔会改 selector，遇到"未找到章节列表"可能是站点结构变了

### 公众号文章特殊说明

- **链接有时效**——分享出来的 URL 几天后可能失效
- 想长期保存的话，**抓下来就别等**
- 公众号会**反爬高频访问**，一次性别抓太多（每天 < 50 篇比较安全）

## 🐛 常见问题

### Q: 报错 "ModuleNotFoundError"
A: 没装依赖。运行 `pip install -r requirements.txt`

### Q: 报错 "未找到章节列表"
A: 笔趣阁的 selector 可能变了，或者那个网站本身结构特殊。可以提供 URL，我帮你看。

### Q: 公众号文章抓下来是乱码
A: 链接可能已经失效。打开 URL 看看浏览器是否能正常显示。

### Q: 字数和实际看到的不一样
A: 我们只统计**中文字符**（不含标点、英文、数字）。这是衡量小说篇幅的行业标准。

### Q: 下载到一半中断了，能继续吗
A: 单本笔趣阁小说目前不支持断点续传。但批量模式下，已经下完的不会重复下，**重新跑就行**。
