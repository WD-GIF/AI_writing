# 提示词模板（你跟我说话的"快捷指令"）

> 复制下面的模板，填空发给我，比纯口语更高效、产出更稳定。

---

## 1️⃣ 写新章节

```
任务：写第 X 章
本章目标：（要发生什么 / 完成什么爽点）
出场人物：A、B、C
场景：（地点、时间、氛围）
本章字数：约 3000 字
结尾要求：（留悬念 / 完成一个小高潮 / 转场到下一场景）
额外要求：（节奏快 / 多对话 / 多打斗描写 / 加心理描写 …）
```

## 2️⃣ 续写 / 接龙

```
任务：接着第 X 章往下写第 X+1 章
重点：（这一章我想推进 XXX）
字数：约 3000 字
```

（我会自动读取上一章的内容保持连贯）

## 3️⃣ 润色 / 改稿

```
任务：润色"章节/第XXX章-XXX.md"
方向：（加强爽感 / 让对话更鲜活 / 节奏再快一点 / 减少水分 / 增强画面感 …）
保留：（哪些剧情/句子不要动）
```

## 4️⃣ 搭大纲

```
任务：帮我搭【分卷大纲 / 章节细纲】
范围：第 X 章 到 第 Y 章
要点：（这一段我想完成 XXX 剧情）
张数：（细到每章一句话 / 细到每章三段）
```

## 5️⃣ 设定扩写

```
任务：帮我把【设定/XXX.md】中的【XXX】部分扩写得更丰满
方向：（更黑暗 / 更细节 / 更有逻辑 / 补一些禁忌和限制 …）
```

## 6️⃣ 人物塑造

```
任务：给【角色名】设计一段【出场戏 / 内心独白 / 关键抉择】
要求：体现这个角色的【XX 性格特质】
长度：约 XXX 字
```

## 7️⃣ 灵感发散

```
任务：给我 X 个【XXX】的点子
约束：（要符合本书世界观 / 要够新颖 / 要避开 XXX 老梗）
```

## 8️⃣ 复盘 / 诊断

```
任务：看看【章节/第XXX章】写得怎么样
诊断角度：（节奏 / 爽感 / 人设一致性 / 伏笔回收 / 文笔）
```

---

## 💡 通用小贴士

- 越具体越好：与其说"写得精彩点"，不如说"主角拔剑的动作慢镜头描写 200 字"
- 不会写就直接说"我不知道，你看着办"，我有默认方案
- 不喜欢的版本告诉我"哪里不喜欢"，我能精准改，避免推倒重来
