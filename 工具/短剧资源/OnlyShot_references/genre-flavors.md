# 流派守则（v0.4.0：加抽象拟人流第 6 流派）

6 大流派，**写作风格完全不同**。阶段 1.1 必须让用户挑流派，挑错后面全错。

> **v0.4.0 新增 F. 抽象拟人流**（来自 SD-002 城市恋综实战 + 2026 水果剧爆款公式拆解）：
> - 真人 100% 演不了的赛道（最高 AI 独占性）
> - 物体 / 概念 / 城市 / 数据 拟人化
> - 拟人 + 狗血 + 极限反转 = 流量公式
> - 视觉风格：软萌赛璐璐 vs 漫剧奇观流的硬朗暗调

---

## A. 红果纯爽流（生活流冲突）

### 特征

- **冲突来自现实利益**：钱、房、孩子、面子
- **当集即爽**：每集 30s 1 个完整爽点循环（被踩 → 反击 → 爽到）
- **人设大于剧情**：观众认的是"嘴贱毒舌但有钱有底牌的奶奶"，不是 30 年前的悬疑
- **生活场景为主**：餐桌、客厅、医院、菜市场（不要太多豪门别墅）
- **对白直白**：不要潜台词太多，观众没空猜

### 节奏

- 前 3 秒：日常被踩（婆婆嫌弃 / 儿媳骂人 / 邻居议论）
- 30 秒：反击爆破（一句话怼回 + 拿出底牌）
- 每分钟 4 个爆点（密度 > 精品悬疑）
- 每集结尾：留一个"明天还有更狠"的钩子

### 代表作（必读拆解）

- 《奶奶觉醒后全家破产了》67 集：奶奶发现儿媳虐待 → 一集一招断财
- 《奶奶的绝地反击》：奶奶联合孙女
- 《有本事别复婚》：女主开挂逆袭，男主追妻不得
- 《家里家外 2》：婆媳大战

### 写作守则

1. **角色少而立体**：4-5 个核心角色，每个有标志性细节（婆婆抠门、女主嘴贱、丈夫憨）
2. **冲突小而具体**：不要 30 年阴谋，要"这周买菜钱被婆婆扣了"
3. **爽点要短**：1 句台词 + 1 个动作 = 1 个完整爽点
4. **避免长线悬疑**：观众不耐烦
5. **背景音乐爆破式**：每个爆点都要鼓点 + 反向音效

### 不要做

- 30 年前家族秘密
- 双胞胎调换
- 退休大法官 / 神医 / 战神
- 复杂的人物关系图
- 长篇内心独白

## B. 精品悬疑流（长线伏笔）

### 特征

- **冲突来自长线阴谋**：30 年前的事 / 家族秘史 / 玄学命格
- **看 N 集等揭晓**：每 5 集一个反转，30 集后真相浮现
- **剧情大于人设**：观众认的是"剧情真有意思"，不是某个角色
- **场景多样化**：豪门、古董店、寺庙、医院、命理馆
- **对白多潜台词**：留白让观众猜

### 节奏

- 前 3 秒：神秘镜头 / 揭秘半句话 / 古怪行为
- 30 秒：第一个伏笔
- 每分钟 3 个爆点（密度低于纯爽，但每个爆点更深）
- 每 5 集大反转，每集小反转
- 第 27、50、67 集是大反转节点

### 代表作

- 《女相师》（听花岛 2026.1）：上古相师为救苍生化十二碎片，弟子下山救世
- 《回廊亭》（陈思诚电影改编）：30 年前家族秘史
- 《云渺》：玄学 + 系列化

### 写作守则

1. **每个角色都有秘密**：6 个角色 = 6 层揭秘
2. **视觉母题密集**：玉佩、八卦镜、信物、刺青
3. **回忆杀必备**：每 3-5 集闪回一次，揭一层
4. **悬念套悬念**：A 反转引出 B 谜团
5. **结局可以开放**：留续集钩子

### 不要做

- 当集即爽（观众等不了 30 集就弃剧）
- 生活流场景为主（要有视觉奇观）
- 直白对白（要有潜台词）
- 简单善恶二元（反派要有动机和挣扎）

### v0.1.0 默认走的就是这个流派

v0.1.0「奶奶断亲：寿宴上的三十年」是精品悬疑流的典型产出。但用户反馈：**这不是红果当下纯爽流**。如果用户要的是红果爆款，应选 A，不是 B。

## C. 漫剧奇观流（AI 漫剧 / 系统 / 末世 / 神话）⭐ AI 优势所在

### 为什么是 AI 真正的护城河

> 用户洞察（v0.1.2 加注）：**真人难拍 = AI 优势所在**。
>
> 真人剧组拍修仙 / 末世 / 玄幻 / 异能 / 西游 / 神话题材，要：
> - 大量特效（炸山 / 喷火 / 飞行 / 异变）
> - 异世界场景搭建（佛塔 / 末世废墟 / 仙山 / 异星）
> - 极端造型（神兽 / 妖魔 / 异族）
> - 不可能的镜头（俯瞰修罗场 / 时空交错 / 神识外放）
>
> 这些**真人剧本来就极贵**或**根本拍不出**。AI 生视频在这些场景下：
> - 不需要剧组 / 群演 / 道具
> - 不需要绿幕特效师
> - 不需要 3D 建模
> - 单段成本 ¥2-5，传统剧组单场可能 ¥10w+
>
> 反过来，奶奶觉醒 / 都市追妻 / 婆媳大战这种**真人能拍**的题材，AI 只是"也能做"，
> 但跟真人比没有显著的成本或质量优势——观众甚至会觉得 AI 演员"假"。
>
> **结论**：用 AI 生视频做短剧，C 流派性价比最高。但本 skill 不强推，让用户自己判断。

### 特征

- **2026 年红果新赛道**：AI 漫剧已超真人占总榜 1/3（《菩提临世真人 AI 版》登顶）
- **市场规模**：DataEye 预测 2026 年漫剧市场 243.6 亿，同比 +45%
- **奇观叙事**：超能力、系统流、末世、神话、修仙、西游
- **视觉冲击 > 剧情**：观众要"震撼画面"，不要复杂剧情
- **人设极端化**：天选之子、系统挂、上古血脉、异能觉醒
- **节奏暴力**：每 30 秒一个奇观点

### 节奏

- 前 3 秒：奇观画面（爆炸 / 飞翔 / 异能 / 怪兽 / 神变）
- 30 秒：异能首次展示（金手指）
- 每分钟 5 个奇观点（密度最高，是 5 流派之最）
- 每集结尾：揭示更大世界观

### 代表作（2026 年红果热榜）

- 《菩提临世真人 AI 版》60 集（2026.4 登顶红果总榜 #1，9000 万热度）
- 《彪悍人生：从制霸高原开始》（5000 万+ 热度）
- 《聚宝仙盆之杂灵根才是真 BOSS》（系统流）
- 《云浮梦若归》（玄幻）
- 《被三叔辜负后我怒闯大海成渔王》（异能）

### 真人难拍场景库（AI 直接生）

写到剧本里时优先选这些，发挥 AI 优势：

| 场景类型 | 例 | 真人成本 | AI 成本 |
|----------|----|---------|--------|
| 修仙飞升 | 主角脚踏祥云、剑光破空 | 极贵（特效）| 单段几元 |
| 末世废墟 | 烧毁的城市、辐射云、丧尸群 | 贵（场景搭）| 单段几元 |
| 异空间 | 时空裂缝、镜像世界、神识入梦 | 极贵 | 单段几元 |
| 神兽 / 妖魔 | 麒麟、龙、九尾狐、骷髅王 | 极贵（CG / 道具）| 单段几元 |
| 异变 / 觉醒 | 主角金光爆发、瞳孔变色、长翅膀 | 贵（特效）| 单段几元 |
| 大型战斗 | 千军万马、剑雨、法术对轰 | 极贵（群演）| 单段几元 |
| 异星 / 异界 | 紫色天空、双月、悬浮岛 | 极贵 | 单段几元 |
| 时间流逝 | 人物速朽 / 万年沧桑变迁 | 极贵 | 单段几元 |

### 写作守则

1. **世界观先行**：第 1 集前 30s 就要交代"这是什么世界"
2. **金手指明确**：主角的能力规则要清楚（什么能做、什么不能做、什么代价）
3. **奇观分级**：第 1 集小奇观（先抓眼）、第 10 集中奇观、第 30 集大奇观、第 50 集史诗级、第 72 集封神
4. **简化对白**：奇观靠画面，对白只交代关键信息（≤ 8 字 / 句更佳）
5. **AI 漫剧 prompt 必带**：指明"漫画风格 / 厚涂 / 二次元光影 / cinematic / cgi"
6. **每集留一个奇观钩子**：结尾揭示下集会出现的更大奇观（"下集出现 ___"）
7. **金手指代价**：能力越强 → 代价越大（防主角无敌）

### 即梦 prompt 关键词（漫剧奇观流专用）

每段 prompt 必带（v0.1.2 jimeng-cli-guide 用）：

```
风格：comic book art, manhua style, cell-shaded, dynamic action lines, vibrant saturated colors,
镜头：dramatic angle, low-angle hero shot / high-angle epic vista, motion blur,
特效：energy aura, glowing eyes, lightning sparks, magic runes, particle effects, lens flare,
氛围：epic, mythical, supernatural, otherworldly,
9:16 vertical, cinematic CGI, ultra detailed, ray traced lighting
```

### 注意

- AI 漫剧审核更严（参考《菩提临世》8 亿播放被下架）
- AIGC 标识必带（红果 2026 强制）
- **不要触碰宗教争议**（菩萨 / 佛祖 / 神仙 / 教派 / 真实历史宗教人物）
- 改用**架空神话体系**（仿西游但不是西游、仿封神但人物名换）
- **不要血腥过度**（红果限制）
- **不要色情擦边**（系统会判定违规）

### 与红果纯爽流的对比（决定怎么选）

| 维度 | A 红果纯爽 | C 漫剧奇观 |
|------|-----------|-----------|
| 真人能拍吗 | 易 | 难 / 极贵 |
| AI 性价比 | 一般 | 高 |
| 观众期待 | 看人 | 看奇观 |
| 剧情复杂度 | 中 | 低 |
| 节奏密度 | 每分钟 4 爆点 | 每分钟 5 奇观点 |
| 角色辨识度要求 | 高（看人）| 中 |
| 市场状态 | 稳定头部 | 上升期黑马 |
| 推荐给 | 想做真人质感的用户 | **用 AI 出片的用户** |

## D. 沙雕轻喜流

### 特征

- **段子密度高**：每集 5-10 个笑点
- **表演夸张**：表情包 / 鬼畜 / 变脸
- **剧情简单**：不需要复杂阴谋
- **梗为主**：网络流行梗 + 反转梗
- **集数偏短**：通常 30-50 集

### 节奏

- 前 3 秒：荒诞动作（如男主开门撞墙）
- 每 30 秒 1 个笑点
- 每集结尾：反差萌钩子（"明天还有更傻的"）

### 代表作

- 《好一个乖乖女》（2025 春节档顶流）

### 写作守则

1. **角色脸谱化**：每个角色一个标签（蠢萌 / 毒舌 / 怂蛋 / 戏精）
2. **段子可独立**：每 30 秒一个笑点，可剪短发抖音
3. **打破第四面墙**：偶尔角色对镜头说话
4. **配乐爆笑**：用熟悉的网络梗配乐
5. **少深刻**：不要试图传达什么思想

### 不要做

- 暗黑悬疑
- 长线伏笔
- 复杂世界观

## E. 年代爽剧流

### 特征

- **背景**：80-90 年代 / 民国 / 建国初期
- **加 buff 设定**：重生 + 年代 / 穿越 + 年代
- **怀旧感 + 爽**：用现代知识降维打击 + 时代红利赚钱
- **女性独立**：女主突破时代束缚
- **服化道讲究**：的确良、暖水瓶、缝纫机

### 节奏

- 前 3 秒：年代场景 + 主角觉醒
- 30 秒：第一次突破时代束缚
- 每分钟 3 个爆点
- 每集结尾：留一个"时代红利"的钩子

### 代表作

- 《我在 80 年代当后妈》（40 集浓缩 80 分钟）
- 《家里家外》系列

### 写作守则

1. **时代细节真实**：粮票、工分、广播站、文工团
2. **年代金句**：用那个年代真实的口头禅（"同志""组织""革命""为人民服务"）
3. **服化道一致性强**：的确良衬衫、灰色棉袄、解放鞋
4. **女主突破设定**：办厂 / 高考 / 出国 / 进城
5. **配乐用年代金曲**：邓丽君 / 蔡琴 / 李谷一

### 不要做

- 现代用语（"内卷""躺平""yyds"）
- 当代场景穿插（除非是穿越前世）
- 与时代背景不符的物件

## 流派选择决策树

skill 阶段 1.1 引导用户挑流派的逻辑：

```
用户原话 → 提取关键词

if "奶奶" in 关键词 and "觉醒" in 关键词:
    建议 A（红果纯爽流），但提醒已饱和

elif "玄学" in 关键词 or "相师" in 关键词 or "命格" in 关键词:
    建议 B（精品悬疑流）

elif "系统" / "末世" / "穿越异世" / "神话" / "修仙" / "西游":
    建议 C（漫剧奇观流），并提醒 AIGC 标识

elif "搞笑" / "沙雕" / "校园":
    建议 D（沙雕轻喜流）

elif "80年代" / "90年代" / "民国" / "重生年代":
    建议 E（年代爽剧流）

else:
    给 5 个选项让用户挑
```

## 流派切换的代价

如果用户在阶段 2/3/4 才意识到选错流派，需要回到阶段 1.1 重选。回退会触发：

- 自动归档到 `归档/v1.X_<日期>_<流派切换前>/`
- 重新跑阶段 1.5、阶段 2.1-2.4
- 角色 / 场景 / 道具圣经可能需要大改

所以**阶段 1.1 必须慎重**。skill 应在用户挑流派前**展示每个流派的代表作 + 当前热度**让用户充分判断。

## F. 抽象拟人流（v0.4.0 新增 ⭐，v0.6.0 加 §0 硬约束）

### ⚠️ 0. ref 工艺偏置硬约束（SD-002 血泪铁律，挑题材前必读）

实测发现：即梦 5.0 对不同物体类型的「拟物化能力」**严重分裂**：

| 拟人对象类型 | 拟物化预期 | 实战表现 |
|--------------|-----------|----------|
| **现代摩天楼 / 玻璃塔** | ✅ 易（5/5） | 上海东方明珠 / 深圳平安金融 / 帝国大厦 → 真正塔身长脸 |
| **简单几何（圆/方/三角）** | ✅ 易（5/5） | 柠檬果实 / 西瓜 / 草莓 / 番茄 → 真正水果长脸 |
| **复杂古建筑** | ⚠️ 难（3/5） | 雷峰塔 / 故宫殿 / 黄鹤楼 / 小蛮腰 / 大雁塔 / 苏州园林 → chibi 人体 + 塔顶头饰 |
| **复杂带文字/图腾物体** | ⚠️ 难（3/5） | 含汉字 logo 的物品 / 含特定图腾的器物 → 模型简化文字成乱码 |

**根因**：即梦 5.0 训练数据偏置。现代摩天楼/简单食物 = 经常被拟人化（Pixar / 国创动画 / 红果短剧），模型学会「物体本身长脸」；复杂古建筑/带文字物 = 训练数据中经常出现在「人物穿戴/cosplay」语境，模型学会「人物穿戴该物」。

**这是 ref 层问题，分镜图层加 `NOT humans wearing X` 子句也无法消除**（实测仅 ~30% 边际改善）。

### ⚠️ 0.1 挑题材避坑清单

```
✅ 推荐选：现代摩天楼 / 简单几何水果 / 卡通食物 / 简化器物 / 抽象数据图标
❌ 避免选：带檐角的古建筑（塔/宫殿/古寺） / 复杂图腾 / 含中文 logo 的物体 / 民族服饰
```

**SD-002 反面教训**：选了 24 城（含杭州雷峰塔 / 北京故宫 / 武汉黄鹤楼 / 广州小蛮腰 / 西安大雁塔 / 苏州园林 / 南京中山陵），其中 8 城是「复杂古建筑」 → 分镜图永远是 chibi 人体戴塔顶头饰，无法纯拟物化。如重选：上海 / 深圳 / 香港中银大厦 / 广州西塔 / 台北 101 / 上海中心大厦 / 北京 CCTV 大裤衩 / 重庆来福士 = 全部现代摩天楼 → 全员纯拟物 5/5。

详见 `references/storyboard-frame-industrial.md` §11 + `references/jimeng-failure-modes.md` §4.3。

### 来源

2026 年 3-5 月 AI 短剧最爆赛道。代表作《Fruit Love Island》TikTok 9 天涨粉 300 万。SD-002「城市恋综」实战沉淀。

### 特征

- **核心**：把非人对象（物体 / 概念 / 城市 / 数据）拟人化当主角
- **AI 独占性**：真人 100% 演不了 → 最高的 AI 护城河
- **狗血 + 拟人 = 流量公式**：套到任何对象都能爆（水果 / 餐具 / 蔬菜 / 手机 / 城市）
- **物体特性深度融入剧情**：榴莲拔壳→学费 / 香蕉皮→脱衣服 / 上海→东方明珠头饰 / 北京→户口本 / 深圳→996 加班代码项链
- **画风**：软萌赛璐璐 / Q 版（避免真实写实，会令人不适）
- **节奏**：1 分钟内极限反转，狗血程度 = 流量

### 适配子题材

| 子题材 | 案例 |

### 写作守则

1. **物体特性 = 剧情燃料**：每个拟人对象的标志符号必须深度融入剧情
   - 榴莲壳 = 财产 / 苹果烂 = 出轨证据 / 西瓜籽 = 怀孕
   - 上海东方明珠 = 光环 / 北京户口 = 婚约 / 深圳房价 = GDP / 杭州直播补光灯 = 暧昧礼物
2. **角色性格 = 刻板印象放大化**（不是侮辱，是「梗」）
   - 上海 = 精致看不起外地 / 深圳 = 996 暴富年轻 / 杭州 = 互联网新贵
3. **狗血戏码强密度**：出轨 / 复仇 / 怀孕 / 离婚 / 绝症 / 失忆，每集 1-2 个
4. **极限反转**：每 30 秒 1 个反转，每集至少 3-4 个
5. **软萌画风**：避免真实写实（会令人不适），用 Q 版 / 赛璐璐 / 卡通
6. **跨次元混搭可加分**：城市 + 武侠 / 餐具 + 修仙 / 蔬菜 + 政治

### 不要做

- 真实写实风格（拟人对象真人化会令人不适）
- 严肃悲剧主题（违背「拟人=笑点」内核）
- 过于复杂的世界观（观众没耐心）
- 单纯翻拍水果剧（红海，必须找新拟人对象）
