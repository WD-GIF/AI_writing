## 时长变奏铁律

每集 36 grid / 4-10s 混合（**不全 5s**）：

```
快切（4s）× 11  → 过渡 / 反应 / 切场
默认（5s）× 14  → 标准对话 / 动作
慢推（6s）× 5   → 爽点 / 反派起疑
特写（7s）× 3   → 重大反转 / 钩子
慢镜（8-10s）× 3 → 心声 / 终极卡点 / 闪回
```

总：~180-195s（不严卡 180）

**为什么变奏**：全 5s 节奏机械感，观众产生「催眠效应」。变奏让关键 beat 强化（10s 慢镜放大冲击）+ 过渡快切（4s 不浪费时间）。

## 红果必爆 7 条铁律

1. **对抗式不自残式**：所有 EP 第 1 个 grid 必须有「外力施压」镜头，不能只是「主角倒下」
2. **金句字幕每集 1 句**：可截图传播的金句（grid 13 / 60-65s 节点）
3. **闪回反差每集 1 次**：现代职场闪回插入神话场景（0.5s 切回，对比强化）
4. **沙雕调剂 ≤ 3 段**：每集搞笑段不超 3 个，避免悬念稀释
5. **结尾倒计时强制**：屏幕底部红色数字（KPI/任务/审判）+ 字幕「未完待续」
6. **角色名前置**：每个新角色登场 grid 必有名牌/字幕/对白点名
7. **时长变奏强制**：4-10s 混合，关键节点放慢（6-10s），过渡放快（4s）
