## 36 宫格分镜（v0.2.0 默认）

| 集长 | 结构 |
|------|------|
| 180-195s | **36 grid × 4-10s 变奏** |
| 120s | 24 grid × 5s（紧凑切片）|
| 90s | 18 grid × 5s（短切片）|

**变奏分布**（默认）：
```
4s × 11 (44s)  → 过渡 / 反应 / 切场
5s × 14 (70s)  → 标准对话 / 动作
6s × 5  (30s)  → 爽点 / 反派起疑
7s × 3  (21s)  → 重大反转 / 钩子
8-10s × 3 (24s) → 心声 / 终极卡点 / 闪回
共 36 段 / 189s
```

**为什么变奏**：全 5s 机械感产生「催眠效应」。变奏让关键 beat 强化（10s 慢镜放大冲击）+ 过渡快切（4s 不浪费时间）。

multimodal2video 单段 4-15s 范围合法，**最低 4s**（不能 3s）。

---

## 60 宫格旧版（已废弃）

v0.1.x 默认 60 grid × 3s。**v0.2.0 后不推荐**：
- 3s 不在 multimodal2video 合法范围（4-15s）
- 60 grid × 11 积分/秒 × 3s = 1980 积分/集，跟 36 grid × 5s 平均成本相同但节奏机械

## 镜头语法

### 景别

| 景别 | 用途 | 短剧典型用法 |
|------|------|-------------|
| 大特写 | 五官 / 物件 | 眼神戏、关键道具特写 |
| 特写 | 头肩 | 对白、表情 |
| 近景 | 半身 | 对白、肢体动作 |
| 中景 | 膝以上 | 走动、互动 |
| 全景 | 全身 + 环境 | 场景建立、群戏 |
| 远景 | 大场景 | 环境氛围 |

**短剧默认偏向**：特写 + 近景为主（手机竖屏看小，远景看不清），关键时刻穿插全景。60 grid 的分布建议：

- 大特写 / 特写：~30 grid（50%）
- 近景 / 中景：~24 grid（40%）
- 全景 / 远景：~6 grid（10%）

### 运镜

| 运镜 | 表达 |
|------|------|
| 固定 | 稳定、客观 |
| 推 | 强调 / 进入情绪 |
| 拉 | 释放 / 揭示更大场面 |
| 摇 | 跟随注视 |
| 移 | 平行跟随 |
| 跟 | 跟随人物 |
| 升 | 拔高、揭示 |
| 降 | 压抑、降临 |

### 角度

| 角度 | 表达 |
|------|------|
| 平视 | 平等 |
| 俯视 | 弱势 / 渺小 |
| 仰视 | 强势 / 崇高 |

## 节奏镜头建议（按 beat 类型）

| beat 类型 | 推荐景别 | 运镜 |
|-----------|----------|------|
| cold_open | 特写 + 推 | 强冲击 |
| satisfaction | 中景 + 慢推 | 让爽点停留 |
| twist | 近景 + 突然推近 | 抓表情变化 |
| pain | 大特写 + 固定 | 让眼泪/血迹定格 |
| cliffhanger | 全景 + 拉 + 黑屏 | 留余韵 |

## 连戏标记（continuity_flags）

每 grid 记录"易出错连戏点"，方便剪辑回查：

```json
"continuity_flags": [
  "苏砚秋: 米色针织衫 + 红丝带（左肩）",
  "林策: 银色打火机（右口袋）",
  "桌上: 一杯咖啡（半满）"
]
```

## 即梦 jimeng_prompt 拼接规则（核心交付！）

每个 grid 的 `jimeng_prompt` 必须按这个模板拼：

```
[视觉风格关键词]，[景别 + 运镜 + 角度]，[场景描述]，[角色描述含 (@角色名_ref.png) 引用 + 动作 + 表情]，[氛围]，[同一角色，服装/发型/外貌不变]，9:16 竖屏，cinematic，电影感，[排除项模板]
```

实例（苏砚秋第 1 集第 1 grid）：

```
高对比戏剧光影，米色暖调 + 暗红点缀，特写慢推，俯视，
古色古香的中式书房，案几上一支毛笔倒下，
(@苏砚秋_ref.png) 苏砚秋坐在案几后，左手摸颈上红丝带（视觉母题），右手握毛笔悬在半空，眼神冷静而锋利，嘴角微微上扬，
氛围：暴风雨前的平静，烛光摇曳，
同一角色，苏砚秋服装/发型/外貌不变，
9:16 竖屏，cinematic，电影感，浅景深，
（排除项）严禁参考图出现在画面中。每个画面为单一画幅，无任何分割线或多宫格效果。No speech bubbles, no comic panels, no split screen, no manga effects, no text overlays, no watermarks, no logos. 表情、嘴型、呼吸、台词严格同步。
```

### 字幕

如有字幕（即梦支持），用中文括号包：

```
（"我不嫁。"）
```

### 角色 ref 引用

每提到一次角色，前面必加 `(@角色名_ref.png)`，无论该角色这格是不是主角。

### 视觉风格后缀必带

每段都要重写完整视觉风格关键词（不要省）。

```markdown
## 故障排查

| 现象 | 应对 |
|------|------|
| 角色脸跳 | 在 prompt 加更多 ref 引用 + 强调「与参考图一致」3 次 |
| 服装跳 | 在 prompt 显式写当前服装颜色 + 款式 |
| 镜头不动 | 强调运镜词：`镜头慢推 / camera dolly in slowly` |
| 出广告水印 | prompt 末尾再加一遍 `no watermarks, no logos` |
| 风格不一致 | 每段都重写完整视觉关键词 |
```
