# 阶段 2：故事架构（节奏地图）深度细则（v0.1.1）

适配 **72 集 × 180s × 60 grid** 的新参数。

---

## 节奏类型表

每个 beat 必属于一种类型。在 `04_节奏地图.json` 用这个枚举：

| 类型 | 含义 | 持续 | 何时用 |
|------|------|------|--------|
| `cold_open` | 黄金 3 秒钩子 | 0-3s | 每集开头第 1 秒 |
| `setup` | 交代背景 | 3-15s | 仅前 3 集 / 新角色登场时 |
| `small_climax` | 小高潮 | 1-3s | 每 15-30s 一次 |
| `twist` | 反转 | 1-3s | 每 30-60s 一次 |
| `satisfaction` | 爽点（公式触发）| 2-5s | 每 30-60s 一次 |
| `pain` | 虐点 | 2-5s | 虐剧每集 1-2 次 |
| `reveal` | 揭秘（信息释放）| 1-3s | 关键节点 |
| `setup_payoff` | 兑现伏笔 | 1-3s | 第 N 集兑现第 M 集埋的 |
| `cliffhanger` | 卡点钩子（结尾）| 5-10s | 每集最后 10 秒必有 |
| `paywall_punch` | 付费点强反转 | 5-10s | 付费短剧第 8/12/26/30/50/55 集结尾 |

## 节奏铁律（每集自查）

按 woshipm 业界硬数据：

1. **第 1 秒就要有动作 / 冲突 / 反差**，不能是字幕、不能是平静镜头
2. 整集 180s，**每分钟 ≥ 3 个情绪爆点**（即整集 ≥ 9 个 small_climax / twist / satisfaction）
3. 整集 ≥ 1 个明确 satisfaction（爽剧 ≥ 3）
4. 30 秒必有第一个爆破点（耳光 + 玻璃碎裂等强冲击）
5. 第 7 分钟（即第 4-5 集结尾）必须有第一个高潮位
6. 结尾 5-10 秒必有 cliffhanger
7. 不能连续 3 集都是同一种节奏类型作主导（会单调）

## 钩子怎么写

### cold_open（每集前 3 秒）

5 种万能开场（任选）：

1. **悬念问句**：黑屏 + 一句独白 → 切镜头
2. **极端动作**：直接砸 / 摔 / 撞 / 扇耳光（**红果首选**）
3. **反差画面**：身份与场景极不匹配
4. **意外反转**：3 秒内一个反转
5. **直白宣战**：角色第一句话就是宣战

woshipm 数据：**捉奸 / 甩脸 / 耳光 / 砸杯子**类强刺激开场，完播率最高。

### cliffhanger（每集最后 5-10 秒）

3 种万能卡点：

1. **真相揭露半句**：眼看要说出真相 → 黑屏
2. **危机即将发生**：危险已经来 → 黑屏
3. **新人物登场**：陌生人推门 → 镜头给脸 → 黑屏

新增（v0.1.1）：

4. **对白半句切断 + 镜头切走**：「其实我——」黑屏
5. **特写定格**：一个表情 / 物件特写 + 字幕「未完待续」

## 情绪曲线

把全 72 集每集情绪强度（0-10）画成曲线。要符合：

- 第 1 集：8（钩子拉满）
- 第 2-7 集：6-7
- 第 8 集（首付费）：9
- 第 9-12 集：7-8
- 第 13-25 集：6-7（缓冲）
- 第 26-30 集：8-9（二付费区）
- 第 31-49 集：6-7（中段冲突）
- 第 50-55 集：9-10（三付费 + 终极反转）
- 第 56-67 集：7-8（兑现伏笔）
- 第 70 集：10（最高）
- 第 71-72 集：8（收尾余韵）

不要让情绪连续 3 集低于 6——观众弃剧。
