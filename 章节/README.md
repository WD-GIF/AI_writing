# 正文章节

## 文件命名规则

```
第001章-章节标题.md
第002章-章节标题.md
...
```

- 三位数编号，前面补 0，方便按文件名排序
- 中间用 `-` 连接章节标题
- 一章一个文件，单章控制在 **2000~3500 字**（网文常规节奏）

## 章节模板

每个章节文件建议这样开头：

```markdown
# 第 X 章 章节标题

> **本章定位**：XX 卷 第 X 章
> **本章字数**：约 XXXX
> **本章功能**：（推进什么剧情 / 完成什么爽点 / 引出什么伏笔）
> **承接**：上一章发生了 XXX
> **悬念**：本章结尾留下 XXX，钩住下一章

---

（正文开始）
```

这样写的好处：日后回看能一眼知道每章在干什么，AI 续写时也能保持连贯。
