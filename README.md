# AI_writing — AI 辅助网文写作工作区

这是一个用于 **AI 辅助网文创作** 的协作仓库。作者负责构思、上传素材与审稿，AI（Cursor Cloud Agent）负责生成、扩写、润色与整理。

## 📁 目录结构

```
AI_writing/
├── 素材/              # 你上传的参考资料（人物原型、设定、灵感、参考文）
├── 设定/              # 世界观、人物设定、金手指、势力关系等
│   ├── 世界观.md
│   ├── 人物设定.md
│   └── 设定补充.md
├── 大纲/              # 故事大纲、卷大纲、章节细纲
│   ├── 总纲.md
│   └── 分卷大纲.md
├── 章节/              # 正文章节（AI 生成的或你自己写的）
│   └── 第001章-示例.md
├── 灵感笔记/          # 临时想法、片段、桥段、金句收集
├── 修改记录/          # 重要修改、版本对比、复盘
└── 工具/              # AI 提示词模板、写作风格指令
    └── 提示词模板.md
```

## 🤝 协作流程

### 1. 你（作者）要做的事

- **上传素材**：把参考资料、设定、灵感扔到对应文件夹（直接拖进 GitHub 网页上传也行）
- **提需求**：告诉我你想要什么，比如：
  - "帮我写第 5 章，主角进入秘境遇到第一个对手"
  - "把这一章润色一下，加强爽感"
  - "根据这个设定写一份 10 万字的分卷大纲"
  - "把人物 A 的性格再丰满一些"
- **审稿**：从仓库下载我生成的内容，自己再修改/发布

### 2. 我（AI）要做的事

- 读取仓库里所有你上传的文件作为创作依据
- 按照你的要求生成内容，**直接写入到仓库对应目录**
- 每次修改都通过 Git 提交，你可以在 GitHub 上看到每一次的变更
- 保持人物、设定、剧情前后一致

## 📥 如何下载我写的内容

三种方式任选：

1. **网页直接看/复制**：在 GitHub 网页上打开 `.md` 文件即可阅读
2. **下载单个文件**：点开文件 → 右上角 "Download raw file"
3. **整个仓库打包**：仓库首页 → 绿色 `Code` 按钮 → `Download ZIP`

## 📤 如何给我上传素材

1. **网页上传**：进入对应文件夹 → `Add file` → `Upload files` → 拖文件进去 → Commit
2. **告诉我链接/内容**：直接把文字发给我，我帮你整理到对应位置

## ✍️ 文件格式建议

- 所有文档统一用 **Markdown** (`.md`) — 轻量、好读、Git 友好
- 章节命名：`第001章-章节标题.md`（三位数编号，方便排序）
- 中文文件名没问题，GitHub 完全支持
